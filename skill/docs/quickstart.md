# 5分钟快速上手

> 从零到写出第一章

## 前提条件

- 已安装 AI IDE（Trae / Claude Code / Cursor 等）
- 已安装 NovelWeaver 技能

## 第一步：初始化项目（2分钟）

### 1.1 输入初始化指令

在 AI IDE 聊天框中输入：

```
/novel-weaver init 开始写一本修仙小说
```

> 你可以替换 "修仙小说" 为你想要的任何题材

### 1.2 选择启动方式

AI 会询问你选择启动方式：

1. **使用预设模板**（推荐新手）- 5个预设模板可选
2. **自定义配置** - 完全控制所有设置
3. **从现有配置导入** - 复用已有配置

### 1.3 完成配置

根据选择的启动方式，完成剩余配置项。AI 会显示进度指示器，让你知道当前进度。

### 1.4 确认配置

完成配置后，AI 会展示配置摘要，确认无误后输入 "确认"。

## 第二步：生成大纲（1分钟）

```
/novel-weaver plan 帮我生成总大纲
```

AI 会生成：
- 三幕结构大纲
- 角色设定
- 世界观框架
- 分卷大纲

## 第三步：写出第一章（2分钟）

```
/novel-weaver write 写第一章，主角在宗门被欺负
```

AI 会自动：
1. 生成 3 个版本的开篇 50 字（黄金开篇）
2. 选择后生成完整章节（2500-3500字）
3. 自动审查质量（AI味检测、一致性检查）
4. 检查字数是否达标

## 第四步：继续写作

```
/novel-weaver write 继续写，主角遇到了敌人
```

### 日常写作循环

```
/novel-weaver write 继续写
  ↓ AI自动: 写作 → 审查 → 字数检查 → 更新大纲/记忆
满意 → 继续
不满意 → /novel-weaver review fix 帮我改一下
字数不足 → /novel-weaver expand 扩充本章
写完当前幕 → /novel-weaver act 下一幕怎么走
```
