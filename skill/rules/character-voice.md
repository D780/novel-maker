# 角色声音检查规则

## 核心原则

每个角色都应该有独特的声音，读者仅通过对话和内心独白就能区分角色。
角色声音 = 语言习惯 + 情感表达 + 知识边界 + 行为逻辑 + 关系互动

---

## 五维度检查体系

### 维度1：语言习惯

**检查点：**

| 特征 | 说明 | 检查方法 |
|------|------|----------|
| 句式长度 | 短句干练 vs 长句复杂 | 统计角色平均句长 |
| 用词偏好 | 口语化 vs 书面化 | 提取高频词 |
| 语气词 | 是否使用"嗯""啊""嘿"等 | 正则匹配 |
| 口头禅 | 标志性表达 | 人工标记 |
| 方言/特殊用语 | 地域或身份标识 | 人工标记 |

**示例对比：**
```
❌ 所有角色都用标准书面语
✅ 战士："干就完了！磨磨唧唧的。"
✅ 学者："此事实则暗含深意，需从长计议。"
✅ 流氓："嘿，你小子，活腻歪了？"
```

### 维度2：情感表达

**检查点：**

| 情感类型 | 不同角色应有不同表达 |
|----------|----------------------|
| 愤怒 | 内敛者沉默 vs 外放者咆哮 |
| 喜悦 | 理性者微笑 vs 感性者欢呼 |
| 悲伤 | 隐忍者独自消化 vs 倾向者寻求安慰 |
| 恐惧 | 冷静者分析 vs 冲动者逃跑 |

**检查方法：**
- 同一情感场景下，不同角色的反应是否一致
- 角色情感变化是否有铺垫

### 维度3：知识边界

**检查点：**

| 维度 | 说明 |
|------|------|
| 专业领域 | 角色只在自己专业内行话流利 |
| 认知水平 | 角色不应使用超出其教育水平的词汇 |
| 信息获取 | 角色不应知道其未接触的信息 |
| 时代限制 | 角色的见识应符合其成长环境 |

**示例：**
```
❌ 农夫使用量子物理术语
✅ 农夫："这天气邪门得很，怕是要变天。"
❌ 从未出村的少女描述巴黎铁塔
✅ 少女："听说城里有高塔，比咱村最高的树还高。"
```

### 维度4：行为逻辑

**检查点：**

| 维度 | 说明 |
|------|------|
| 性格一致性 | 角色行为不应违背其核心性格 |
| 决策模式 | 理性者分析利弊 vs 感性者凭直觉 |
| 习惯动作 | 标志性小动作（摸鼻子、转笔） |
| 底线与原则 | 角色在压力下仍坚守什么 |

**检查方法：**
- 关键决策时，角色的选择是否符合其人设
- 是否有"为了剧情需要"而让角色做出违背性格的行为

### 维度5：关系互动

**检查点：**

| 关系类型 | 互动特征 |
|----------|----------|
| 上下级 | 敬语 vs 命令语气 |
| 朋友 | 随意玩笑 vs 真诚关心 |
| 敌人 | 针锋相对 vs 暗中较劲 |
| 恋人 | 亲密称呼 vs 特殊默契 |

**检查方法：**
- 同一角色对不同人的说话方式是否不同
- 关系变化后，互动方式是否相应调整

## 使用方式

### 写作时检查

1. **对话标签化**
   ```markdown
   <!-- 角色：林风（战士） -->
   <!-- 语言：短句、口语、偶尔脏话 -->
   <!-- 当前情感：愤怒 -->
   ```

2. **声音对照表**
   | 角色 | 语言习惯 | 情感表达 | 知识边界 |
   |------|----------|----------|----------|
   | 林风 | 短句、口语 | 直接、外放 | 武术、江湖 |
   | 苏雨 | 长句、书面 | 内敛、含蓄 | 医术、诗词 |

### 审查时检查

```markdown
## 角色声音审查清单

### 语言习惯
- [ ] 角色平均句长是否符合设定
- [ ] 是否有标志性口头禅
- [ ] 用词风格是否一致

### 情感表达
- [ ] 情感反应是否符合性格
- [ ] 是否有情感铺垫
- [ ] 不同角色反应是否区分

### 知识边界
- [ ] 是否有超出认知的表达
- [ ] 信息来源是否合理
- [ ] 专业术语使用是否恰当

### 行为逻辑
- [ ] 关键决策是否符合性格
- [ ] 是否有违背人设的行为
- [ ] 底线是否坚守

### 关系互动
- [ ] 对不同人说话方式是否不同
- [ ] 关系变化后互动是否调整
```

## 常见问题

### 1. 角色声音同质化

**问题：** 所有角色听起来都一样

**解决：**
- 为每个角色创建"声音档案"
- 限制角色词汇表
- 增加标志性表达

### 2. 角色行为突变

**问题：** 角色突然做出违背性格的事

**解决：**
- 增加心理铺垫
- 设置触发条件（如极端压力）
- 通过他人反应暗示异常

### 3. 信息越界

**问题：** 角色知道不该知道的事

**解决：**
- 建立信息流向图
- 标记信息获取节点
- 通过对话自然传递信息
