# 反AI表达规则

> 检测并修正AI写作中的"AI味"，让小说更像真人写的。整合原 anti-ai-patterns.md、anti-ai-techniques.md。

## 一、7层检测体系

### 第1层：广告词过度

**检测模式：**
- 连续3个以上形容词修饰同一名词
- 使用"完美"、"极致"、"无与伦比"等绝对化词汇
- 使用"仿佛"、"宛如"、"恰似"等比喻词过度（每章超过3次）

**修正：** 删除冗余形容词，保留最有力的1-2个；用具体描写替代抽象形容。

### 第2层：情感标签化

**检测模式：**
- "他感到愤怒"、"她心中一喜"等直接情感陈述
- "气氛变得紧张"等氛围标签
- "一股暖流涌上心头"等陈词滥调

**修正：** 用动作展示情感（"他猛地拍了一下桌子"）；用环境烘托氛围。

### 第3层：对话形式化

**检测模式：**
- 对话中使用"您"过多（除非角色设定需要）
- 对话句子过长（超过30字）
- 对话中使用书面语（"然而"、"因此"、"尽管如此"）
- 对话缺乏口语化表达（嗯、啊、呢、吧、嘛）

**修正：** 增加口语化表达；缩短对话句子；添加打断、犹豫、重复。

### 第4层：结构闭合化

**检测模式：**
- 段落结尾用"总之"、"总而言之"、"由此可见"
- 章节结尾用"这一切，只是开始"等套路
- 每个场景结束后都有"反思段"

**修正：** 删除总结性语句；用悬念或行动结尾；留白。

### 第5层：转场公式化

**检测模式：**
- "时间一晃，来到了..."、"转眼间..."
- "与此同时，在另一个地方..."
- "另一边，XXX也在..."

**修正：** 直接切入新场景；用环境变化暗示时间流逝；用角色行动自然转场。

### 第6层：总结倾向

**检测模式：**
- "接下来的几天里..."、"在接下来的日子里..."
- "经过一番努力..."、"经过长时间的..."
- "终于，在XXX的帮助下..."

**修正：** 展示具体过程，而非总结结果；选取关键场景详细描写。

### 第7层：信息曝光

**检测模式：**
- 角色突然知道不该知道的信息
- 旁白透露过多背景信息
- 对话中塞入过多设定

**修正：** 控制信息释放节奏；用角色视角限制信息；通过事件自然揭示。

## 二、AI高频词汇速查

### 禁用句式

| 套话类型 | 示例 | 替代方案 |
|---------|------|---------|
| 连续"了"字 | "他看了看，想了想，说了说" | "他端详片刻，斟酌着开口" |
| 连续"的"字 | "蓝色的天空的下面" | "蓝天下" |
| "不禁"开头 | "不禁感到" | 直接描写感受 |
| "仿佛"滥用 | "仿佛...一般" | 具体描写 |
| "一丝"滥用 | "一丝笑意" | 具体描写表情 |
| "一抹"滥用 | "一抹苦涩" | 直接描述情感 |

### 高频词汇（谨慎使用）

- **情感类：** 苦涩、复杂、微妙、难以言喻、心中涌起、内心深处、五味杂陈
- **动作类：** 缓缓、默默、轻轻、深吸一口气、苦笑一声、皱了皱眉
- **描写类：** 犹如、宛若、好似、映入眼帘、传入耳中、不知何时
- **惊讶套路：** 竟然、居然、没想到
- **模糊表达：** 似乎、好像、仿佛

## 三、深层去AI味技巧

### 1. 角色对话区分度

**核心原则：** 遮住角色名字，能认出是谁说的。

**方法：** 为每个角色设计独特的说话方式（参见 [rules/character-voice.md](character-voice.md)）；加入语气词、口头禅；根据性格调整句式长短。

```
角色A（粗犷）："干他娘的！"
角色B（文雅）："此事...不妥。"
角色C（急躁）："快！快！快！"
```

### 2. 情感表达行为化

```
❌ "他很愤怒"
✅ "他攥紧了拳头，指节发白"

❌ "她感到悲伤"
✅ "她低下头，眼泪无声地滑落"
```

### 3. 感官细节丰富化

不止视觉，加入听觉、嗅觉、味觉、触觉：

```
❌ "他走进了一个房间"
✅ "他推开门，一股霉味扑面而来，脚下是吱呀作响的木地板"
```

### 4. 句式长短变化

- 紧张场景：短句、断句（"跑！快跑！来不及了。"）
- 舒缓场景：长句、复合句
- 对话场景：根据角色性格调整

### 5. 金句与名场面

每章至少一个"金句"或"名场面"。金句要简洁有力、富有哲理、符合角色性格。

## 四、常见AI-slop模式（60+）

### 广告词类
完美无瑕、无与伦比、登峰造极、独一无二、前所未有、惊天动地、气势磅礴、精妙绝伦

### 情感标签类
心中一喜、心中一怒、心中一惊、感到一阵温暖、气氛变得紧张、一股暖流涌上心头、不由得、不禁

### 对话套路类
"你..."他欲言又止。/ "哼！"他冷哼一声。/ "呵..."他轻笑一声。/ "哦？"他挑了挑眉。

### 结构套路类
这一切，只是开始。/ 命运的齿轮，开始转动。/ 故事，才刚刚开始。/ 这一战，注定载入史册。

### 转场套路类
时间一晃，来到了三天后。/ 与此同时，在另一个地方.../ 转眼间，一个月过去了。

### 描写套路类
他的眼神变得深邃起来。/ 他的嘴角微微上扬。/ 他的眼中闪过一丝精光。/ 他的气势瞬间暴涨。

### 总结套路类
总之，这一切都结束了。/ 由此可见，他的实力不容小觑。/ 毫无疑问，这是一场胜利。

## 五、自检清单

### 词汇检查
- [ ] 是否使用了"不禁"、"竟然"、"一抹"等AI高频词？
- [ ] 是否连续使用了"淡淡"、"缓缓"、"轻轻"？
- [ ] 是否过度使用了"眼眸"、"嘴角"、"眉头"？

### 句式检查
- [ ] 是否连续3句以上使用相同句式？
- [ ] 是否有"一股...的感觉"的模糊表达？
- [ ] 是否有"仿佛...一般"的陈词滥调？

### 对话检查
- [ ] 遮住名字，能认出每个角色吗？
- [ ] 对话是否过于正式/书面化？

### 情感检查
- [ ] 情感表达是通过行为而非直述吗？
- [ ] 情感变化有过渡吗？

### 结构检查
- [ ] 是否有总结性语句？
- [ ] 是否有公式化转场？
- [ ] 是否有信息泄露？

### 亮点检查
- [ ] 本章是否有金句或名场面？
- [ ] 是否有让读者印象深刻的瞬间？

## 六、平台特殊要求

| 平台 | 要点 |
|------|------|
| 番茄小说 | 每章必须有钩子；爽点密度高（每3章至少1个）；避免大段心理描写 |
| 起点中文网 | 世界观展开合理；升级节奏稳；伏笔要回收；避免过于直白的爽文套路 |
| 晋江文学城 | 情感线细腻；角色魅力突出；主题表达深刻；避免过于商业化 |

## 七、使用方式

- **写作时：** `根据 anti-ai-expressions.md 的规则，避免使用广告词和情感标签`
- **审查时：** `/novel-maker review 检查第5章的AI味`
- **脚本辅助：** `python scripts/auditor/style_check.py 本章.md --json`（自动检测AI高频词密度）
