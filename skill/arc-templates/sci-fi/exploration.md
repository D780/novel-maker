# 星际探索弧

> 人类首次深空探索，经历漫长航行和未知危机，最终完成历史性发现

## 弧线结构

启程 → 航行 → 发现 → 危机 → 突破 → 归途

## 阶段定义

### 阶段 1：启程（2-3章）
**目标：** 建立探索的使命和团队，展现出发时的壮阔
**关键事件：**
- 探索计划的背景（为何必须远航）
- 主角加入团队的原因和动机
- 发射时刻的震撼与告别
**情绪走向：** 期待 → 壮烈
**章节建议：** 2-3章

### 阶段 2：航行（2-3章）
**目标：** 展现太空航行的漫长与孤独
**关键事件：**
- 飞船内部的日常生活和团队磨合
- 通信延迟带来的与地球的疏离
- 小型技术故障，考验团队协作
**情绪走向：** 平静 → 孤独
**章节建议：** 2-3章

### 阶段 3：发现（2-3章）
**目标：** 抵达目标区域，发现超出预期的现象
**关键事件：**
- 到达目的地，开始探测任务
- 发现异常信号或不明物体
- 初步接触，震撼与困惑并存
**情绪走向：** 兴奋 → 震撼
**章节建议：** 2-3章

### 阶段 4：危机（2-3章）
**目标：** 探索引发重大危机，团队面临生死考验
**关键事件：**
- 接触不明存在导致飞船系统故障
- 团队成员受伤或牺牲
- 与地球失联，孤立无援
**情绪走向：** 恐惧 → 绝望
**章节建议：** 2-3章

### 阶段 5：突破（1-2章）
**目标：** 找到解决问题的关键，化解危机
**关键事件：**
- 主角从发现中找到突破口
- 利用新知识修复系统或化解威胁
- 成功与未知存在建立沟通或理解
**情绪走向：** 希望 → 坚定
**章节建议：** 1-2章

### 阶段 6：归途（1-2章）
**目标：** 带着收获踏上返程，留下深远影响
**关键事件：**
- 收集关键数据，准备返航
- 团队反思这次探索的意义
- 预示人类未来的新篇章
**情绪走向：** 感慨 → 希望
**章节建议：** 1-2章

## 关键场景

1. **发射时刻**：火箭升空，主角望向渐远的地球，告别熟悉的一切
2. **首次接触**：发现不明存在的瞬间，科学仪器疯狂报警，团队震惊
3. **太空行走修复**：危机中主角亲自出舱修复关键设备，在宇宙中感受渺小

## 注意事项

- **太空真实感**：失重、辐射、通信延迟等细节要真实
- **团队动态**：封闭空间中的人际关系是重要看点
- **节奏张弛**：航行阶段不能太无聊，用小事件维持张力
- **未知的敬畏**：外星发现要保持神秘感，不要过度解释

## 适用题材

- 太空歌剧、硬科幻、探索冒险
- 适合描写人类勇气、未知恐惧、文明碰撞
