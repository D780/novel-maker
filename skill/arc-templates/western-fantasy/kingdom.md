# 王国争霸弧

> 主角从底层崛起，经历扩张、战争与危机，最终统一大陆的帝王弧线

## 弧线结构

崛起 → 扩张 → 战争 → 危机 → 统一

## 阶段定义

### 阶段 1：崛起（3-4章）
**目标：** 主角从微末中崛起，建立自己的势力根基
**关键事件：**
- 主角出身低微但拥有非凡才能或特殊血统
- 通过某次事件崭露头角，获得第一批追随者
- 占据一处根据地，开始积累资源和人才
**情绪走向：** 隐忍 → 爆发 → 野心
**章节建议：** 3-4章

### 阶段 2：扩张（3-4章）
**目标：** 势力范围不断扩大，引起其他王国的注意
**关键事件：**
- 通过军事或外交手段吞并周边势力
- 建立行政体系，招揽文臣武将
- 与其他王国产生利益摩擦，暗流涌动
**情绪走向：** 得意 → 警惕 → 霸气
**章节建议：** 3-4章

### 阶段 3：战争（3-4章）
**目标：** 多方势力全面开战，主角面临最严峻的军事考验
**关键事件：**
- 多个王国联合对抗主角，形成围攻之势
- 大规模战争场面，策略与武力的双重较量
- 内部出现叛徒或分裂，腹背受敌
**情绪走向：** 紧张 → 惨烈 → 咬牙坚持
**章节建议：** 3-4章

### 阶段 4：危机（2-3章）
**目标：** 胜利在望时遭遇致命危机，跌入最低谷
**关键事件：**
- 关键战役惨败或遭遇超自然力量的干预
- 最信任的人背叛或牺牲
- 主角陷入绝望，面临放弃或孤注一掷的抉择
**情绪走向：** 震惊 → 绝望 → 绝地反击
**章节建议：** 2-3章

### 阶段 5：统一（2-3章）
**目标：** 从低谷中崛起，完成最终统一
**关键事件：**
- 重新集结力量，制定最终战略
- 决定性战役，一战定乾坤
- 建立新秩序，面对统一后的治理挑战
**情绪走向：** 坚毅 → 激昂 → 沉甸甸的胜利
**章节建议：** 2-3章

## 关键场景

1. **初露锋芒**：主角第一次展现军事才能或领导力的战斗场景，需要有"此人不可小觑"的震撼感
2. **背叛时刻**：最信任的人倒戈的场景，需要有强烈的情感冲击和戏剧反转
3. **最终决战**：决定大陆归属的史诗级大战，需要有宏大的战争场面和细腻的个人英勇

## 注意事项

- **主角不能全靠金手指**：崛起要靠智慧和努力，不能全靠血脉觉醒或天降神器
- **战争要有策略**：不能写成"人多就赢"，要有真实的战术博弈
- **统一不是终点**：打天下容易治天下难，结局要暗示治理的挑战
- **配角要有立场**：追随者不能无脑忠诚，要有自己的利益考量和选择

## 适用题材

西方奇幻、帝国崛起、王国战争、权力的游戏、帝王传记
