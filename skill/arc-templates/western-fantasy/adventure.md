# 冒险弧

> 主角踏上未知旅途，结识伙伴，历经挑战，最终完成伟大冒险的史诗弧线

## 弧线结构

启程 → 伙伴 → 挑战 → 成长 → 决战 → 归来

## 阶段定义

### 阶段 1：启程（2-3章）
**目标：** 建立主角的平凡起点，触发踏上冒险之旅的契机
**关键事件：**
- 主角身处平凡环境，但内心渴望远方或肩负使命
- 某个事件打破日常（收到任务/发现秘密/家园遭难）
- 告别熟悉的一切，踏上未知旅途
**情绪走向：** 平静 → 动荡 → 决意
**章节建议：** 2-3章

### 阶段 2：伙伴（2-3章）
**目标：** 在旅途中结识志同道合的伙伴，建立团队
**关键事件：**
- 遇到性格迥异的未来伙伴（战士/法师/盗贼/牧师等）
- 通过共同经历小规模危机建立信任
- 团队成员各有秘密或个人目标，为后续冲突埋下伏笔
**情绪走向：** 孤独 → 温暖 → 归属
**章节建议：** 2-3章

### 阶段 3：挑战（2-3章）
**目标：** 面对旅途中的重重考验，团队经受磨合
**关键事件：**
- 穿越危险地域（黑暗森林/荒漠/地下城）
- 与敌对势力发生正面冲突
- 团队内部出现分歧或信任危机
**情绪走向：** 紧张 → 疲惫 → 坚定
**章节建议：** 2-3章

### 阶段 4：成长（2-3章）
**目标：** 通过历练获得实力和心智的双重成长
**关键事件：**
- 主角发现并掌握新的能力或武器
- 解开伙伴之间的误会，团队羁绊加深
- 获得关于最终目标的关键线索或道具
**情绪走向：** 沉淀 → 蜕变 → 信心
**章节建议：** 2-3章

### 阶段 5：决战（2-3章）
**目标：** 与最终敌人或障碍展开决定性对决
**关键事件：**
- 到达最终目的地，揭示敌人的真正面目
- 惨烈的战斗，团队成员面临生死考验
- 主角发挥成长所得，以团队之力战胜强敌
**情绪走向：** 紧张 → 惨烈 → 激昂
**章节建议：** 2-3章

### 阶段 6：归来（1-2章）
**目标：** 冒险结束，主角带着收获和改变回归
**关键事件：**
- 战后余波，疗伤与告别
- 主角回到起点，但已不是当初的自己
- 留下开放性暗示，为新冒险埋下种子
**情绪走向：** 释然 → 感慨 → 新的期待
**章节建议：** 1-2章

## 关键场景

1. **启程之刻**：主角站在家乡边界回望的场景，需要有"回不去"的决绝感和对未知的期待
2. **伙伴集结**：团队成员第一次并肩作战的场景，需要展现各人特长和配合默契
3. **最终决战**：全员协力对抗终极威胁的高潮场景，需要每人都有发挥，体现团队力量

## 注意事项

- **伙伴不能是工具人**：每个伙伴都要有独立的性格弧线和成长，不能只在需要时出场
- **挑战要层层递进**：难度必须逐步提升，不能一开始就是最高难度然后一路平推
- **成长要有代价**：能力提升必须伴随牺牲或付出，否则成长缺乏分量
- **归来不能草率**：冒险结束后的收尾同样重要，要给读者情感上的满足和闭环

## 适用题材

西方奇幻、史诗冒险、地下城探索、奇幻公路片、勇者传说
