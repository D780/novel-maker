# 门派争霸弧

> 主角带领门派从微末崛起，经历挑战与危机，最终统一武林的争霸弧线

## 弧线结构

崛起 → 挑战 → 危机 → 决战 → 统一

## 阶段定义

### 阶段 1：崛起（3-4章）
**目标：** 主角接手或创立门派，从弱小开始发展壮大
**关键事件：**
- 主角成为门派掌门或核心人物（继承/临危受命/自立门户）
- 门派弱小，资源匮乏，弟子稀少
- 通过几次小规模冲突证明实力，吸引人才加入
**情绪走向：** 艰难 → 坚韧 → 初见成效
**章节建议：** 3-4章

### 阶段 2：挑战（3-4章）
**目标：** 门派崛起引起其他势力的注意和打压
**关键事件：**
- 大门派对主角门派发出挑战或威胁
- 参加武林大会或门派比武，名震江湖
- 门派内部出现分歧——是依附强者还是独立发展
**情绪走向：** 紧张 → 激昂 → 露锋芒
**章节建议：** 3-4章

### 阶段 3：危机（2-3章）
**目标：** 门派遭遇灭顶之灾，主角面临最大考验
**关键事件：**
- 多个门派联手围攻，或遭遇魔教入侵
- 门派弟子伤亡惨重，根据地被毁或被围
- 主角身负重伤，门派前途未卜
**情绪走向：** 震惊 → 惨烈 → 绝境
**章节建议：** 2-3章

### 阶段 4：决战（2-3章）
**目标：** 主角绝地反击，以弱胜强扭转局势
**关键事件：**
- 主角在绝境中领悟绝学或获得奇遇
- 重新集结残部，联合盟友发动反攻
- 与敌方首领展开巅峰对决
**情绪走向：** 隐忍 → 爆发 → 荡气回肠
**章节建议：** 2-3章

### 阶段 5：统一（1-2章）
**目标：** 门派成为武林领袖，建立新秩序
**关键事件：**
- 击败主要对手，门派声望达到顶峰
- 召开武林大会，建立新的江湖秩序
- 主角面对统一后的责任与取舍
**情绪走向：** 豪迈 → 沉稳 → 胸怀天下
**章节建议：** 1-2章

## 关键场景

1. **临危受命**：主角在门派危难之际挺身而出的场景，需要有"我不入地狱谁入地狱"的气概
2. **武林扬名**：主角在公开比武中一战成名的场景，需要有以弱胜强的震撼和观众的惊叹
3. **绝地反击**：门派在覆灭边缘发起反攻的场景，需要有悲壮感和绝处逢生的燃点

## 注意事项

- **门派发展要有细节**：不能一笔带过"门派壮大了"，要有收弟子、建据点、练武功的具体描写
- **不能只靠主角一人**：门派争霸是集体事业，配角和弟子都要有存在感和贡献
- **武功提升要合理**：主角变强要有修炼过程，不能靠"打通任督二脉"一步到位
- **统一要付出代价**：争霸过程中的牺牲和取舍要有分量，不能全员happy ending

## 适用题材

武侠、门派经营、武林争霸、江湖政治、掌门成长
