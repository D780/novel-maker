# 江湖恩怨弧

> 主角因恩怨纠葛卷入江湖纷争，经历追杀与反击，最终了结恩仇的弧线

## 弧线结构

相遇 → 冲突 → 追杀 → 反击 → 了结

## 阶段定义

### 阶段 1：相遇（2-3章）
**目标：** 建立恩怨的起点，交代人物关系的渊源
**关键事件：**
- 主角与对手因某种缘分相遇（比武/救命/同门）
- 建立初始关系（师兄弟/挚友/救命恩人）
- 暗藏矛盾的种子——利益、理念或情感上的分歧
**情绪走向：** 平和 → 亲近 → 暗流
**章节建议：** 2-3章

### 阶段 2：冲突（2-3章）
**目标：** 矛盾激化，关系破裂，恩怨正式形成
**关键事件：**
- 某个事件引爆积累的矛盾（背叛/误杀/夺宝）
- 双方立场对立，曾经的情谊化为仇恨
- 江湖中人各站阵营，恩怨牵连越来越广
**情绪走向：** 震惊 → 愤怒 → 决裂
**章节建议：** 2-3章

### 阶段 3：追杀（2-3章）
**目标：** 主角陷入被动，遭受追杀和围堵
**关键事件：**
- 对手势力庞大，主角被全面追杀
- 主角被迫逃亡，沿途遭遇截杀
- 逃亡中得到意外帮助（隐世高人/忠义之士）
**情绪走向：** 压抑 → 疲于奔命 → 隐忍
**章节建议：** 2-3章

### 阶段 4：反击（2-3章）
**目标：** 主角积蓄力量，由守转攻
**关键事件：**
- 在逃亡中武功精进，领悟新招式
- 找到对手的弱点或关键证据
- 联合被对手迫害的其他势力，策划反击
**情绪走向：** 沉淀 → 冷静 → 凌厉
**章节建议：** 2-3章

### 阶段 5：了结（1-2章）
**目标：** 恩怨最终清算，主角做出选择
**关键事件：**
- 与对手正面对决，揭开所有真相
- 战斗中发现恩怨背后更大的阴谋
- 了结恩怨——是杀是放，主角做出艰难抉择
**情绪走向：** 肃杀 → 释然 → 感慨
**章节建议：** 1-2章

## 关键场景

1. **决裂时刻**：曾经亲密的两人反目成仇的场景，需要有强烈的情感撕裂感
2. **逃亡截杀**：主角在险境中被追杀的场景，需要有紧迫感和绝境求生的紧张
3. **最终对决**：恩怨双方的宿命之战，需要有武打的精彩和情感的纠葛交织

## 注意事项

- **恩怨要有分量**：仇恨的起因必须足够重大且有说服力，不能是鸡毛蒜皮的小事
- **对手不能是纯恶**：对手要有自己的立场和苦衷，让读者能理解其行为动机
- **追杀不能太轻松**：逃亡过程要有真实的危险感，主角不能每次都化险为夷
- **了结要留余味**：恩怨了结后要有反思，不能打完就完，要让读者有所思考

## 适用题材

武侠、江湖恩怨、门派斗争、复仇记、侠客行
