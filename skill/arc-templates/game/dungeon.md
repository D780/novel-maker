# 副本弧

> 玩家团队挑战高难度副本，经历开荒失败，研究攻略，最终通关的经典游戏体验

## 弧线结构

发现 → 准备 → 开荒 → 团灭 → 攻略 → 通关

## 阶段定义

### 阶段 1：发现（1章）
**目标：** 发现新副本，建立挑战目标
**关键事件：**
- 新副本开放或主角发现隐藏副本
- 了解副本背景和奖励，产生兴趣
- 组建挑战队伍，制定初步计划
**情绪走向：** 兴奋 → 期待
**章节建议：** 1章

### 阶段 2：准备（1-2章）
**目标：** 为开荒做充分准备
**关键事件：**
- 收集副本情报，研究怪物机制
- 准备装备、药水、消耗品
- 队伍磨合，分配职责（T/DPS/治疗）
**情绪走向：** 紧张 → 期待
**章节建议：** 1-2章

### 阶段 3：开荒（2-3章）
**目标：** 首次挑战，摸索机制
**关键事件：**
- 进入副本，感受氛围
- 遭遇小怪和第一个BOSS，初尝胜利
- 遇到机制复杂的BOSS，开始受挫
**情绪走向：** 紧张 → 激烈
**章节建议：** 2-3章

### 阶段 4：团灭（1-2章）
**目标：** 反复失败，团队士气低落
**关键事件：**
- 关键BOSS团灭，装备耐久耗尽
- 队员互相指责，产生矛盾
- 有人提议放弃，团队面临解散
**情绪走向：** 沮丧 → 绝望
**章节建议：** 1-2章

### 阶段 5：攻略（2-3章）
**目标：** 重新研究，找到通关方法
**关键事件：**
- 主角分析录像，找到失败原因
- 调整战术，重新分配任务
- 反复练习，逐步推进
**情绪走向：** 希望 → 专注
**章节建议：** 2-3章

### 阶段 6：通关（1-2章）
**目标：** 最终击败BOSS，通关副本
**关键事件：**
- 最终决战，全员配合默契
- 关键时刻的神操作或意外
- 击杀BOSS，获得丰厚奖励
**情绪走向：** 紧张 → 狂喜
**章节建议：** 1-2章

## 关键场景

1. **初见BOSS**：BOSS登场的震撼，全队被秒杀的绝望
2. **团灭时刻**：即将成功时突然团灭，功亏一篑的痛苦
3. **通关瞬间**：BOSS倒地，全队欢呼，掉落稀有装备

## 注意事项

- **机制描述**：BOSS机制要清晰有趣，不要过于复杂
- **团队配合**：强调团队协作，个人英雄主义要适度
- **失败价值**：每次团灭都要有收获，不能单纯重复
- **节奏把控**：开荒过程要有进展感，不能原地踏步

## 适用题材

- 网游、游戏异界、虚拟现实
- 适合描写团队合作、策略对抗、挑战极限
