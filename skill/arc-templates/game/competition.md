# 竞技弧

> 主角参加游戏竞技比赛，从预赛到决赛，一路过关斩将，最终夺冠的热血历程

## 弧线结构

报名 → 预赛 → 复赛 → 半决赛 → 决赛 → 夺冠

## 阶段定义

### 阶段 1：报名（1章）
**目标：** 建立参赛动机，展示比赛规模
**关键事件：**
- 竞技比赛开放报名
- 主角决定参赛，可能有特殊原因
- 展示比赛的规模和奖励
**情绪走向：** 期待 → 决心
**章节建议：** 1章

### 阶段 2：预赛（1-2章）
**目标：** 通过预赛，展示基础实力
**关键事件：**
- 预赛规则介绍
- 主角轻松晋级，展示基础操作
- 遇到第一个有威胁的对手
**情绪走向：** 轻松 → 紧张
**章节建议：** 1-2章

### 阶段 3：复赛（2-3章）
**目标：** 遭遇强敌，开始认真对待
**关键事件：**
- 对手实力明显提升
- 主角被迫使用压箱底技巧
- 险胜晋级，意识到比赛难度
**情绪走向：** 紧张 → 激烈
**章节建议：** 2-3章

### 阶段 4：半决赛（2-3章）
**目标：** 遭遇宿敌或最强对手之一
**关键事件：**
- 遇到宿敌或夺冠热门
- 对手研究过主角的战术
- 被逼入绝境，绝地反击
**情绪走向：** 激烈 → 激动
**章节建议：** 2-3章

### 阶段 5：决赛（2-3章）
**目标：** 最终决战，巅峰对决
**关键事件：**
- 决赛前的准备和心理战
- 决赛开始，双方全力以赴
- 关键时刻的神操作或意外
**情绪走向：** 紧张 → 震撼
**章节建议：** 2-3章

### 阶段 6：夺冠（1章）
**目标：** 获得冠军，实现目标
**关键事件：**
- 获得冠军，全场欢呼
- 颁奖典礼，接受采访
- 回顾比赛历程，展望未来
**情绪走向：** 狂喜 → 满足
**章节建议：** 1章

## 关键场景

1. **宿命对决**：半决赛遭遇宿敌，两人眼神交锋，全场屏息
2. **绝地反击**：决赛陷入绝境，主角爆发潜能，逆转局势
3. **夺冠时刻**：胜利瞬间，奖杯举起，全场沸腾

## 注意事项

- **比赛公平**：比赛规则要清晰公平，不要靠作弊取胜
- **对手塑造**：对手要有魅力，不能是单纯的反派
- **技术描写**：比赛过程要有技术含量，不能只靠运气
- **赛后影响**：夺冠要有意义，影响后续剧情发展

## 适用题材

- 网游、游戏异界、电竞题材
- 适合描写竞技对抗、巅峰对决、追梦历程
