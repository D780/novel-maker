# 篇章弧线模板

> 为卷级规划提供结构化的弧线模板

## 使用方式

在幕规划时，选择合适的弧线模板：

```
/novel-maker act 使用挑战弧模板规划下一幕
```

## 弧线类型

### 通用弧线（6种）
- 挑战弧：准备→挑战→失败→成长→再战→胜利
- 探索弧：发现→探索→危险→收获→揭秘
- 冲突弧：摩擦→升级→对决→后果→新平衡
- 悬疑弧：线索→调查→误导→真相→反转
- 成长弧：困境→学习→实践→突破→蜕变
- 关系弧：相遇→冲突→理解→和解→深化

### 题材特定弧线
- 修仙：境界突破弧、宗门大比弧、秘境探险弧、渡劫弧
- 都市：商战弧、复仇弧、崛起弧、感情弧
- 无限流：副本弧、强化弧、团队弧、BOSS战弧
- 悬疑：案件弧、推理弧、反转弧
- 历史：权谋弧、战役弧、改革弧
- 科幻：科技突破弧、星际探索弧、AI觉醒弧、文明接触弧
- 游戏：升级弧、副本弧、公会战弧、竞技弧
- 末世：生存弧、势力争霸弧、异能觉醒弧、人性考验弧

## 弧线模板结构

每个弧线模板包含：
- **阶段定义**：每个阶段的目标和关键事件
- **章节建议**：每个阶段的建议章节数
- **情绪曲线**：每个阶段的情绪走向
- **关键场景**：必须包含的关键场景
- **注意事项**：常见错误和避坑指南
