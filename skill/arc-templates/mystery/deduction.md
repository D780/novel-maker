# 推理弧

> 主角面对一个谜题，通过收集信息、分析推理、提出假设并验证的弧线

## 弧线结构

谜题 → 收集 → 分析 → 假设 → 验证

## 阶段定义

### 阶段 1：谜题（2-3章）
**目标：** 抛出谜题，建立推理的目标和动机
**关键事件：**
- 出现一个令人困惑的谜题或反常现象
- 主角被谜题吸引或被迫参与
- 初步观察后发现情况远比想象复杂
**情绪走向：** 好奇 → 困惑
**章节建议：** 2-3章

### 阶段 2：收集（2-3章）
**目标：** 系统性地收集与谜题相关的信息
**关键事件：**
- 走访相关人员，收集口供和证物
- 查阅历史资料、文献或记录
- 发现一些看似无关但可能重要的细节
**情绪走向：** 耐心 → 专注
**章节建议：** 2-3章

### 阶段 3：分析（2-3章）
**目标：** 对收集到的信息进行深度分析和整合
**关键事件：**
- 将零散的信息进行分类和关联
- 发现信息之间的矛盾和不合理之处
- 剔除干扰信息，提炼核心要素
**情绪走向：** 沉思 → 曙光初现
**章节建议：** 2-3章

### 阶段 4：假设（1-2章）
**目标：** 基于分析提出合理的假设或理论
**关键事件：**
- 提出一个或多个可能的解释
- 用已知信息检验假设的合理性
- 遇到无法解释的反例，修正假设
**情绪走向：** 兴奋 → 犹疑
**章节建议：** 1-2章

### 阶段 5：验证（1-2章）
**目标：** 通过实际行动验证假设，得出最终结论
**关键事件：**
- 设计验证方案，收集决定性证据
- 揭示真相，验证或推翻假设
- 真相大白后的反思和总结
**情绪走向：** 紧张 → 豁然开朗
**章节建议：** 1-2章

## 关键场景

1. **谜题初现**：谜题首次暴露的场景，需要足够引人入胜，让读者也想一探究竟
2. **灵光一闪**：主角在分析过程中灵光乍现的场景，需要有清晰的思维链
3. **真相验证**：验证假设的高潮场景，需要有决定性的证据和逻辑闭环

## 注意事项

- **推理过程要有代入感**：让读者能跟着主角一起思考，而不是旁观主角的天才推理
- **假设要经得起推敲**：每个假设都应该有逻辑支撑，不能靠猜
- **避免事后诸葛亮**：验证环节的信息不能是全新的，必须是之前线索的合理推演
- **推理不等于暴力破解**：不要让主角靠穷举法或运气得出结论

## 适用题材

推理小说、数学谜题、密码破解、历史悬案、科学探秘、逻辑游戏
