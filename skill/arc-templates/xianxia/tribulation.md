# 渡劫弧

> 主角面对天劫考验，经历失败后重修，最终成功渡劫的弧线

## 弧线结构

准备 → 渡劫 → 失败 → 再修 → 成功

## 阶段定义

### 阶段 1：准备（2-3章）
**目标：** 渡劫在即，主角进行全面准备
**关键事件：**
- 感知到天劫将至，确定大致时间
- 搜集渡劫所需的丹药、法宝和防护手段
- 向有经验的前辈请教渡劫心得
**情绪走向：** 紧迫 → 郑重
**章节建议：** 2-3章

### 阶段 2：渡劫（1-2章）
**目标：** 天劫降临，主角正面应对
**关键事件：**
- 天劫正式降临，雷劫轰下
- 主角运用准备的手段逐一抵挡
- 天劫威力远超预期，形势急转直下
**情绪走向：** 紧张 → 危急
**章节建议：** 1-2章

### 阶段 3：失败（1-2章）
**目标：** 渡劫失败，遭受重创，但保住性命
**关键事件：**
- 在最后关头功亏一篑，被天劫击溃
- 身受重伤，修为受损
- 内心遭受巨大打击，但求生意志未灭
**情绪走向：** 绝望 → 低谷
**章节建议：** 1-2章

### 阶段 4：再修（2-3章）
**目标：** 从失败中汲取教训，重新修炼
**关键事件：**
- 疗伤恢复，反思失败的原因
- 发现自身功法或心境的缺陷
- 针对性地弥补不足，修炼新的保命手段
**情绪走向：** 沉淀 → 坚定
**章节建议：** 2-3章

### 阶段 5：成功（1-2章）
**目标：** 再次面对天劫，成功渡过
**关键事件：**
- 天劫再次降临，这次更加猛烈
- 主角吸取上次教训，从容应对
- 在最后关头以全新领悟成功渡劫
**情绪走向：** 紧张 → 狂喜
**章节建议：** 1-2章

## 关键场景

1. **天劫降临**：雷劫轰下的宏大场面，需要有天地之威的压迫感
2. **功亏一篑**：渡劫失败的惨烈场景，需要让读者感到痛心但不绝望
3. **逆天成功**：再次渡劫成功的壮丽场景，需要与上次失败形成鲜明对比

## 注意事项

- **失败不能白费**：第一次失败必须有实质性的收获（认知/经验），否则就是注水
- **再修不能太长**：重修过程要精炼，重点放在认知提升而非重复修炼
- **天劫要有层次**：不同阶段的天劫威力应有递进，不能每次都一样
- **心境比实力重要**：渡劫的核心考验应该是心境而非单纯的实力对抗

## 适用题材

修仙渡劫、武道突破、飞升成仙、炼体锻魂、逆天改命
