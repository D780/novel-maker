# 宗门大比弧

> 主角参加宗门内部比武大会，一路过关斩将，最终赢得荣耀的弧线

## 弧线结构

报名 → 初赛 → 复赛 → 决赛 → 奖励

## 阶段定义

### 阶段 1：报名（2-3章）
**目标：** 交代大比背景、规则和参赛动机，建立期待感
**关键事件：**
- 宗门大比即将开始，公布规则和奖励
- 主角决定参赛（主动或被迫），明确目标
- 各方势力暗流涌动，潜在对手陆续登场
**情绪走向：** 期待 → 紧张
**章节建议：** 2-3章

### 阶段 2：初赛（2-3章）
**目标：** 通过初赛展示实力，同时观察潜在对手
**关键事件：**
- 初赛采用群战/积分/淘汰制，门槛较低
- 主角轻松通过，但有意隐藏部分实力
- 观察到几个实力强劲的对手，暗中评估
**情绪走向：** 从容 → 留意
**章节建议：** 2-3章

### 阶段 3：复赛（2-3章）
**目标：** 遭遇强劲对手，比赛进入白热化
**关键事件：**
- 遇到实力相当的对手，展开苦战
- 被迫展露更多底牌和手段
- 险胜或惨胜，同时暴露了弱点
**情绪走向：** 紧张 → 激烈
**章节建议：** 2-3章

### 阶段 4：决赛（2-3章）
**目标：** 巅峰对决，大比的最高潮
**关键事件：**
- 对阵最强对手，双方倾尽全力
- 战斗过程中展现各自的成长和底牌
- 关键时刻以出人意料的方式取胜
**情绪走向：** 白热化 → 震撼
**章节建议：** 2-3章

### 阶段 5：奖励（1-2章）
**目标：** 领取奖励，收获名声，为后续剧情铺垫
**关键事件：**
- 获得大比奖励（功法/丹药/资源/地位）
- 名声大振，引来各方关注（包括敌意）
- 大比中发现的线索或伏笔开始展开
**情绪走向：** 满足 → 展望
**章节建议：** 1-2章

## 关键场景

1. **强者亮相**：主要对手首次展示实力的震撼场景，需要让读者感受到威胁
2. **苦战获胜**：主角遭遇真正强敌、险胜的场景，需要节奏紧凑、攻防转换精彩
3. **决赛巅峰**：大比最终对决的场景，需要调动所有铺垫，做到情绪和战斗的双重高潮

## 注意事项

- **对手不能太弱**：每个阶段的对手都要有真正的威胁，不能让主角碾压全场
- **战斗要多样化**：不同对手应有不同的战斗风格和手段，避免千篇一律的打斗
- **不要忽略配角**：其他参赛者的战斗和故事也能丰富大比的精彩度
- **大比要有后续影响**：大比的结果必须对后续剧情产生实质性影响，不能比完就忘

## 适用题材

修仙宗门、武道大会、学院竞技、比武招亲、天下第一武道会
