# 战役弧

> 主角面对一场重大战役，经历备战和苦战，最终取得胜利的弧线

## 弧线结构

备战 → 交锋 → 苦战 → 转折 → 胜利

## 阶段定义

### 阶段 1：备战（2-3章）
**目标：** 展示战前形势，进行战略部署
**关键事件：**
- 分析敌我双方的实力对比和战略态势
- 制定作战计划，进行军事部署
- 战前的紧张气氛和士兵的心态
**情绪走向：** 严肃 → 紧迫
**章节建议：** 2-3章

### 阶段 2：交锋（2-3章）
**目标：** 战役正式打响，双方初次交手
**关键事件：**
- 先锋部队交战，试探敌方实力
- 小规模胜利或失利，调整战略
- 发现敌方的战术意图或隐藏手段
**情绪走向：** 紧张 → 激烈
**章节建议：** 2-3章

### 阶段 3：苦战（2-3章）
**目标：** 战役进入最残酷的阶段
**关键事件：**
- 双方主力正面交锋，伤亡惨重
- 我方陷入不利局面（被包围/补给断裂/援军不到）
- 战场上出现令人动容的英勇或牺牲
**情绪走向：** 惨烈 → 悲壮
**章节建议：** 2-3章

### 阶段 4：转折（1-2章）
**目标：** 战场形势发生决定性逆转
**关键事件：**
- 主角发现敌方的致命破绽
- 执行一个大胆的战术或奇谋
- 关键时刻的援军/资源/信息到达
**情绪走向：** 绝望边缘 → 希望重燃
**章节建议：** 1-2章

### 阶段 5：胜利（1-2章）
**目标：** 取得战役的最终胜利
**关键事件：**
- 全面反攻，敌方溃败
- 清点战果和伤亡
- 胜利后反思战争的残酷和意义
**情绪走向：** 激昂 → 沉重中的欣慰
**章节建议：** 1-2章

## 关键场景

1. **战前动员**：大战前的誓师或激励场景，需要有感染力和使命感
2. **苦战惨烈**：战斗最激烈、最残酷的场景，需要有真实感和情感冲击
3. **决胜时刻**：扭转战局的关键场景，需要有节奏感和大场面

## 注意事项

- **战争要有谋略**：不能只是两军对冲，需要有战略和战术的较量
- **伤亡要真实**：战争必然伴随死亡，不能主角这边零伤亡
- **不要神化主角**：主角可以优秀但不能无敌，胜利应该有团队的贡献
- **反思战争**：胜利后应该有对战争代价的反思，不能只是庆祝

## 适用题材

古代战争、军事小说、抗战、三国争霸、草原征战、海战
