# 生存弧

> 末日灾难降临，主角从求生到适应，经历危机与稳定，最终带领幸存者重建文明

## 弧线结构

灾难 → 求生 → 适应 → 稳定 → 危机 → 重建

## 阶段定义

### 阶段 1：灾难（1-2章）
**目标：** 灾难降临，世界崩塌
**关键事件：**
- 灾难发生（丧尸/病毒/核战/天灾）
- 主角目睹世界崩溃，被迫逃离
- 失去亲人或熟悉的一切
**情绪走向：** 震惊 → 恐惧
**章节建议：** 1-2章

### 阶段 2：求生（2-3章）
**目标：** 在废墟中挣扎求生
**关键事件：**
- 寻找食物、水、庇护所
- 遭遇其他幸存者，结盟或冲突
- 学习末世生存技能
**情绪走向：** 恐惧 → 焦虑
**章节建议：** 2-3章

### 阶段 3：适应（2-3章）
**目标：** 逐渐适应末世环境，建立生存模式
**关键事件：**
- 建立临时据点
- 形成小团队，分工合作
- 建立资源收集和防御体系
**情绪走向：** 焦虑 → 平静
**章节建议：** 2-3章

### 阶段 4：稳定（2-3章）
**目标：** 生存稳定，开始思考未来
**关键事件：**
- 据点发展，人口增加
- 与其他幸存者群体建立联系
- 开始种植、生产，恢复部分秩序
**情绪走向：** 平静 → 希望
**章节建议：** 2-3章

### 阶段 5：危机（2-3章）
**目标：** 重大危机威胁生存成果
**关键事件：**
- 外敌入侵（掠夺者/变异生物/自然灾害）
- 内部矛盾爆发（权力斗争/资源争夺）
- 主角面临艰难抉择
**情绪走向：** 紧张 → 绝望
**章节建议：** 2-3章

### 阶段 6：重建（1-2章）
**目标：** 克服危机，开启重建之路
**关键事件：**
- 主角带领众人战胜危机
- 重建更稳固的社区
- 展望人类文明的新篇章
**情绪走向：** 希望 → 坚定
**章节建议：** 1-2章

## 关键场景

1. **灾难降临**：世界崩塌的震撼瞬间，主角逃离城市
2. **第一次杀戮**：为了生存被迫伤害他人，内心的挣扎
3. **重建希望**：在废墟中种下第一颗种子，象征新生

## 注意事项

- **生存细节**：食物、水、医疗等资源要真实
- **人性黑暗**：末世中的人性考验要深刻
- **希望平衡**：不能过于绝望，要保持希望
- **领导成长**：主角从普通人到领导者的成长要自然

## 适用题材

- 末日生存、废土、后启示录
- 适合描写人性考验、生存挣扎、文明重建
