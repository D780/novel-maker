# 势力争霸弧

> 末世中各方势力崛起，经历扩张与冲突，最终决出霸主，统一废土

## 弧线结构

崛起 → 扩张 → 冲突 → 危机 → 决战 → 统一

## 阶段定义

### 阶段 1：崛起（2-3章）
**目标：** 主角势力从无到有，初步建立
**关键事件：**
- 主角带领小团队建立据点
- 吸纳流民，扩充人口
- 击退小股掠夺者，建立威望
**情绪走向：** 谨慎 → 坚定
**章节建议：** 2-3章

### 阶段 2：扩张（2-3章）
**目标：** 势力范围扩大，资源增加
**关键事件：**
- 占领更多据点，控制资源
- 建立贸易网络，与其他势力结盟
- 组建武装力量，训练士兵
**情绪走向：** 野心 → 热血
**章节建议：** 2-3章

### 阶段 3：冲突（2-3章）
**目标：** 与其他势力产生冲突，摩擦升级
**关键事件：**
- 边境冲突，资源争夺
- 外交谈判破裂
- 小规模战斗爆发
**情绪走向：** 紧张 → 敌意
**章节建议：** 2-3章

### 阶段 4：危机（2-3章）
**目标：** 势力面临重大危机
**关键事件：**
- 强敌压境，联盟破裂
- 内部叛变或暗杀阴谋
- 资源枯竭或天灾来袭
**情绪走向：** 焦虑 → 绝望
**章节建议：** 2-3章

### 阶段 5：决战（2-3章）
**目标：** 与最强对手的最终决战
**关键事件：**
- 双方集结全部力量
- 史诗级大战
- 关键时刻的转折点
**情绪走向：** 紧张 → 激烈
**章节建议：** 2-3章

### 阶段 6：统一（1-2章）
**目标：** 成为废土霸主，建立新秩序
**关键事件：**
- 战胜最后对手，统一废土
- 建立新政权，制定法律
- 开启重建时代
**情绪走向：** 骄傲 → 使命感
**章节建议：** 1-2章

## 关键场景

1. **第一次扩张**：主角带领队伍攻下第一个敌对据点
2. **联盟破裂**：昔日盟友背叛，主角陷入孤立
3. **最终决战**：两大势力的终极对决，决定废土命运

## 注意事项

- **势力设计**：各方势力要有特色，不能脸谱化
- **战争描写**：战斗场面要有策略性，不是单纯的人数堆砌
- **政治权谋**：末世政治要复杂真实
- **道德困境**：争霸过程中的道德选择要深刻

## 适用题材

- 末日争霸、废土、后启示录
- 适合描写势力对抗、权谋斗争、领导艺术
