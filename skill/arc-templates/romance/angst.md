# 虐恋弧

> 男女主角深爱彼此却因误会分离，经历痛苦与思念，最终重逢和好的虐心弧线

## 弧线结构

相遇 → 相爱 → 误会 → 分离 → 重逢 → 和好

## 阶段定义

### 阶段 1：相遇（2-3章）
**目标：** 建立男女主角的相遇，为后续的深爱做铺垫
**关键事件：**
- 男女主角相遇，彼此留下深刻印象
- 因共同经历逐渐走近
- 暗示两人之间存在某种阻碍（身份差距/家族恩怨/前世纠葛）
**情绪走向：** 好奇 → 心动 → 甜蜜中带隐忧
**章节建议：** 2-3章

### 阶段 2：相爱（2-3章）
**目标：** 展现两人深爱的过程，让读者投入这段感情
**关键事件：**
- 经历生死考验，确认彼此的心意
- 深入了解对方，感情迅速升温
- 许下承诺，以为可以一直在一起
**情绪走向：** 甜蜜 → 深情 → 幸福到顶点
**章节建议：** 2-3章

### 阶段 3：误会（2-3章）
**目标：** 制造令人心碎的误会，将甜蜜撕裂
**关键事件：**
- 有人蓄意制造误会（栽赃/欺骗/断章取义）
- 当事人因性格原因不愿解释或无法解释
- 误会不断扩大，信任崩塌
**情绪走向：** 震惊 → 不肯相信 → 心碎
**章节建议：** 2-3章

### 阶段 4：分离（2-3章）
**目标：** 两人被迫分开，各自承受痛苦
**关键事件：**
- 其中一方提出分手或不告而别
- 分离后各自的生活——表面坚强内心痛苦
- 偶然得知对方的消息，或在远处默默关注
**情绪走向：** 撕裂 → 麻木 → 深深的思念
**章节建议：** 2-3章

### 阶段 5：重逢（1-2章）
**目标：** 命运让两人再次相遇，旧情复燃
**关键事件：**
- 在某个场合意外重逢
- 发现彼此都未能放下对方
- 真相逐渐浮出水面，误会开始化解
**情绪走向：** 震惊 → 激动 → 犹豫
**章节建议：** 1-2章

### 阶段 6：和好（1-2章）
**目标：** 解开所有误会，重新走到一起
**关键事件：**
- 完整的真相大白，误会彻底消除
- 双方坦诚内心，承认分离期间的痛苦
- 重新在一起，感情比之前更加珍惜和牢固
**情绪走向：** 释然 → 泪水 → 重获幸福
**章节建议：** 1-2章

## 关键场景

1. **深爱时刻**：两人相爱最深的场景，需要有让读者感到"这就是真爱"的感染力
2. **误会爆发**：误会导致信任崩塌的场景，需要有令人心碎的情感冲击
3. **重逢瞬间**：分离后再次相遇的场景，需要有复杂的情感交织——惊喜、心酸、犹豫

## 注意事项

- **误会必须有合理原因**：不能是"你为什么不听我解释"的低级误会，要有无法解释的客观原因
- **虐要虐到点上**：虐心不是靠狗血桥段堆砌，而是靠真实的情感困境和人物选择
- **分离不能太久**：分离阶段要控制篇幅，避免读者失去耐心
- **和好要有诚意**：不能一句"对不起"就翻篇，要有深入的情感沟通和和解过程

## 适用题材

言情、虐恋、爱情悲剧、破镜重圆、深情男主/女主
