# 追妻火葬场弧

> 男主角不懂珍惜导致失去女主角，经历后悔与追妻，最终破镜重圆的弧线

## 弧线结构

抛弃 → 后悔 → 追妻 → 考验 → 复合

## 阶段定义

### 阶段 1：抛弃（2-3章）
**目标：** 展现男主角对女主角的忽视和伤害，建立读者的愤怒
**关键事件：**
- 男主角对女主角的付出视而不见，或心有白月光
- 在关键时刻做出伤害女主角的选择（偏向他人/否认关系/当众羞辱）
- 女主角心灰意冷，决定离开
**情绪走向：** 心寒 → 绝望 → 决绝
**章节建议：** 2-3章

### 阶段 2：后悔（2-3章）
**目标：** 女主角离开后，男主角开始意识到失去的珍贵
**关键事件：**
- 女主角离开后，男主角才发现她一直在默默付出
- 日常生活中处处是女主角的痕迹，才明白她的重要性
- 得知女主角过得越来越好，开始慌张
**情绪走向：** 不习惯 → 意识到 → 恐慌
**章节建议：** 2-3章

### 阶段 3：追妻（3-4章）
**目标：** 男主角开始追妻之路，但女主角已不是当初的她
**关键事件：**
- 男主角放下身段追求，但女主角态度冷淡
- 女主角变得独立自信，不再围着男主角转
- 男主角为了挽回做出各种努力（道歉/改变/证明自己），但屡屡碰壁
**情绪走向：** 焦急 → 挫败 → 不放弃
**章节建议：** 3-4章

### 阶段 4：考验（2-3章）
**目标：** 追妻过程中遭遇真正的考验，验证男主角的改变
**关键事件：**
- 女主角身边出现新的追求者，男主角面临竞争
- 女主角遭遇困难，男主角挺身而出但被拒绝帮助
- 男主角为了女主角做出重大牺牲或改变
**情绪走向：** 嫉妒 → 煎熬 → 坚定
**章节建议：** 2-3章

### 阶段 5：复合（1-2章）
**目标：** 女主角看到男主角的真心改变，选择再给一次机会
**关键事件：**
- 某个关键时刻，男主角用行动证明了自己的改变
- 女主角内心防线瓦解，承认自己从未真正放下
- 两人重新在一起，但这次是平等的、互相珍惜的关系
**情绪走向：** 释然 → 感动 → 圆满
**章节建议：** 1-2章

## 关键场景

1. **心寒离开**：女主角决定离开的场景，需要有"终于醒悟"的决绝和令人心疼的坚强
2. **后悔独白**：男主角意识到失去的场景，需要有对比——女主角在时vs不在后的巨大落差
3. **追妻碰壁**：男主角追妻被冷脸相对的场景，需要有"当初你不要，现在追不回"的爽感

## 注意事项

- **抛弃阶段要控制度**：男主角的伤害不能太过分（家暴/出轨），否则读者无法接受复合
- **后悔要真实**：后悔不能来得太快太突然，要有意识到的过程
- **女主角不能太软**：追妻火葬场的爽点在于女主角不再卑微，要保持她的独立和尊严
- **复合要有门槛**：不能一追就回，要有足够的考验让读者觉得"他配得上了"

## 适用题材

言情、追妻火葬场、先虐后甜、破镜重圆、霸道总裁
