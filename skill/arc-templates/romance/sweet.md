# 甜蜜恋爱弧

> 主角从相遇到相爱，经历甜蜜与考验，最终走到一起的甜蜜恋爱弧线

## 弧线结构

相遇 → 心动 → 表白 → 热恋 → 考验 → 结合

## 阶段定义

### 阶段 1：相遇（2-3章）
**目标：** 建立男女主角的初次相遇，埋下缘分的种子
**关键事件：**
- 男女主角以某种方式相遇（误会/意外/命中注定）
- 第一印象不佳或各怀心思
- 因某种原因不得不继续接触（工作/学业/任务）
**情绪走向：** 好奇 → 误解 → 好奇加深
**章节建议：** 2-3章

### 阶段 2：心动（2-3章）
**目标：** 在相处中逐渐发现对方的闪光点，产生好感
**关键事件：**
- 在日常相处中发现对方不为人知的温柔或才华
- 某个瞬间被对方深深打动（英雄救美/体贴细节/才华展现）
- 开始在意对方的一举一动，但嘴上不承认
**情绪走向：** 甜蜜 → 小鹿乱撞 → 患得患失
**章节建议：** 2-3章

### 阶段 3：表白（1-2章）
**目标：** 感情明朗化，双方确认心意
**关键事件：**
- 某个契机让隐藏的感情暴露（酒后吐真言/生死关头/第三者刺激）
- 正式表白，甜蜜互动
- 关系确立，周围人的反应（祝福/嫉妒/反对）
**情绪走向：** 紧张 → 狂喜 → 甜蜜
**章节建议：** 1-2章

### 阶段 4：热恋（2-3章）
**目标：** 展现恋爱中的甜蜜与幸福
**关键事件：**
- 恋爱日常——约会、互送礼物、甜蜜互动
- 互相深入了解，分享彼此的过去和梦想
- 感情不断升温，认定对方是此生挚爱
**情绪走向：** 幸福 → 甜蜜 → 沉浸
**章节建议：** 2-3章

### 阶段 5：考验（2-3章）
**目标：** 感情遭遇外部或内部的考验
**关键事件：**
- 外部阻力——家庭反对/距离分隔/第三者介入
- 内部矛盾——性格差异/信任危机/误会
- 面临选择——事业与爱情的冲突
**情绪走向：** 焦虑 → 痛苦 → 挣扎
**章节建议：** 2-3章

### 阶段 6：结合（1-2章）
**目标：** 克服考验，感情更加牢固，走到一起
**关键事件：**
- 双方坦诚沟通，化解误会和矛盾
- 共同面对外部压力，携手克服困难
- 正式在一起，展望共同的未来
**情绪走向：** 释然 → 感动 → 圆满
**章节建议：** 1-2章

## 关键场景

1. **命中相遇**：男女主角初次相遇的场景，需要有戏剧性和记忆点，让人感到"缘分天注定"
2. **心动瞬间**：主角被对方打动的场景，需要有细腻的心理描写和氛围营造
3. **甜蜜日常**：恋爱中的甜蜜互动场景，需要有生活化的细节和让人会心一笑的温馨

## 注意事项

- **甜不能硬撒**：甜蜜要自然，通过细节和互动展现，不能写成"两人对视就脸红心跳"
- **考验不能太虐**：这是甜蜜恋爱弧，考验要适度，不能写成虐恋
- **配角不能抢戏**：情敌和家人是催化剂，不能喧宾夺主
- **感情要有层次**：从陌生到心动到深爱，每个阶段的感受要有区别

## 适用题材

言情、甜宠、校园恋爱、都市恋爱、浪漫喜剧
