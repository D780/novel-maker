# 日久生情弧

> 男女主角从普通关系开始，在长期相处中逐渐发现彼此的好，最终走到一起的弧线

## 弧线结构

相识 → 相处 → 心动 → 表白 → 结合

## 阶段定义

### 阶段 1：相识（2-3章）
**目标：** 建立男女主角的初始关系，明确"非恋爱"的起点
**关键事件：**
- 男女主角以非浪漫方式相识（同事/邻居/青梅竹马/契约关系）
- 彼此之间没有恋爱的感觉，甚至有些看不顺眼
- 因为客观原因不得不长期相处
**情绪走向：** 无所谓 → 小摩擦 → 习惯
**章节建议：** 2-3章

### 阶段 2：相处（3-4章）
**目标：** 在日常相处中逐渐了解对方，建立默契
**关键事件：**
- 日常生活中的小互动——一起吃饭/互相帮忙/分享烦恼
- 在对方遇到困难时伸出援手，发现对方不为人知的一面
- 建立起只有两人之间才有的默契和暗号
**情绪走向：** 习惯 → 舒适 → 依赖
**章节建议：** 3-4章

### 阶段 3：心动（2-3章）
**目标：** 某个瞬间意识到自己对对方的感情已经变了
**关键事件：**
- 看到对方与异性亲近时感到不舒服（吃醋但不自知）
- 某个瞬间被对方深深打动（认真工作的样子/不经意的温柔/突然的保护）
- 开始在意自己的形象，想在对方面前表现更好
**情绪走向：** 困惑 → 确认 → 慌张
**章节建议：** 2-3章

### 阶段 4：表白（1-2章）
**目标：** 承认感情，捅破窗户纸
**关键事件：**
- 在某个契机下感情暴露（醉酒/意外/被朋友点破）
- 正式表白——可能是双向暗恋的揭晓
- 面对"万一表白失败连朋友都做不成"的恐惧
**情绪走向：** 紧张 → 坦诚 → 甜蜜
**章节建议：** 1-2章

### 阶段 5：结合（1-2章）
**目标：** 从朋友变恋人，关系自然过渡
**关键事件：**
- 刚在一起时的小别扭——不知道如何从朋友切换到恋人
- 朋友们的反应（早就看出来了/大吃一惊）
- 找到属于两人的恋爱方式，感情稳定而深厚
**情绪走向：** 甜蜜 → 调整 → 安稳的幸福
**章节建议：** 1-2章

## 关键场景

1. **习惯养成**：两人不知不觉中养成只有彼此才有的习惯的场景，需要有润物细无声的温暖
2. **醋意觉醒**：主角看到对方与异性亲近时感到不舒服的场景，需要有"后知后觉"的可爱
3. **心意相通**：表白时发现原来是双向暗恋的场景，需要有"原来你也喜欢我"的甜蜜

## 注意事项

- **慢要慢得有内容**：每章都要有推进关系的细节，不能原地踏步
- **不能太慢**：日久生情不等于拖沓，要有节奏感，让读者感到"终于要在一起了"的期待
- **朋友感要到位**：两人在一起前的默契和信任感是这个弧线的灵魂，必须写足
- **自然过渡**：从朋友到恋人不能太突兀，要有情感积累的水到渠成感

## 适用题材

言情、日久生情、青梅竹马、先婚后爱、办公室恋爱、室友恋爱
