# 挑战弧

> 主角面对强大挑战，经历失败后成长，最终战胜对手的经典弧线

## 弧线结构

准备 → 挑战 → 失败 → 成长 → 再战 → 胜利

## 阶段定义

### 阶段 1：准备（2-3章）
**目标：** 建立挑战的重要性，展示主角当前实力与目标的差距
**关键事件：**
- 明确挑战的目标和赌注（失败的代价）
- 主角评估自身实力，发现差距
- 制定策略或开始针对性训练
**情绪走向：** 期待 → 紧张
**章节建议：** 2-3章

### 阶段 2：挑战（1-2章）
**目标：** 正面迎接挑战，暴露主角的不足
**关键事件：**
- 挑战正式开始
- 主角尝试用现有能力应对
- 逐步发现对手远超预期
**情绪走向：** 紧张 → 激烈
**章节建议：** 1-2章

### 阶段 3：失败（1章）
**目标：** 主角遭受挫折，制造低谷
**关键事件：**
- 主角失败或被迫撤退
- 承受失败的代价（受伤/失去重要事物/被人嘲笑）
- 内心动摇，但未放弃
**情绪走向：** 挫败 → 低落
**章节建议：** 1章

### 阶段 4：成长（2-3章）
**目标：** 从失败中学习，找到突破口
**关键事件：**
- 反思失败原因，找到关键弱点
- 获得提升方法（新技能/新认知/新盟友）
- 刻苦训练或积累资源
**情绪走向：** 希望 →
