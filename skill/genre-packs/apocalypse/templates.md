## 场景模板

### 末世场景
```markdown
## 场景设置
- 地点：（城市废墟/避难所/野外）
- 环境：（天气/时间/危险等级）
- 资源状况：

## 场景流程
1. 进入：进入场景
2. 探索：搜索资源或信息
3. 遭遇：遇到危险或机遇
4. 应对：采取行动
5. 发现：关键发现
6. 撤离：安全离开或陷入困境

## 描写要点
- 环境的荒凉和危险
- 资源的稀缺和珍贵
- 人性的善恶和挣扎
- 生存的艰难和希望

## 情绪曲线
- 进入：紧张→谨慎
- 探索：好奇→警觉
- 遭遇：震惊→恐惧
- 应对：紧张→焦虑
- 发现：惊喜→希望
- 撤离：释放→期待
```

### 生存场景
```markdown
## 场景设置
- 地点：（避难所/野外营地/安全屋）
- 时间：（白天/夜晚）
- 资源状况：

## 生存流程
1. 准备：准备生存物资
2. 执行：执行生存任务
3. 意外：突发状况
4. 应对：处理危机
5. 收获：获得资源或信息
6. 休整：恢复和整理

## 生存要素
- 食物获取
- 水源寻找
- 避难所维护
- 防御工事

## 情绪曲线
- 准备：紧张→专注
- 执行：专注→疲惫
- 意外：震惊→焦虑
- 应对：紧张→坚定
- 收获：惊喜→满足
- 休整：平静→期待
```

### 势力冲突场景
```markdown
## 场景设置
- 地点：（势力边界/资源点/避难所）
- 冲突双方：
- 冲突原因：

## 冲突流程
1. 对峙：双方对峙
2. 交涉：尝试谈判
3. 破裂：谈判破裂
4. 战斗：爆发冲突
5. 转折：关键变化
6. 结局：冲突结果

## 冲突要素
- 实力对比
- 资源争夺
- 理念冲突
- 人性考验

## 情绪曲线
- 对峙：紧张→戒备
- 交涉：谨慎→希望
- 破裂：失望→愤怒
- 战斗：激烈→恐惧
- 转折：震惊→紧张
- 结局：释放→深思
```
