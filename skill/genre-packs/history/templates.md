## 场景模板

### 宫廷场景
```markdown
## 场景设置
- 地点：（大殿/御书房/后宫/花园）
- 时间：（早朝/午后/深夜）
- 氛围：（庄严/紧张/轻松/诡异）

## 参与人物
- 核心人物：
- 配角人物：
- 暗中观察者：

## 场景流程
1. 开场：场景设定和人物就位
2. 铺垫：看似平常的对话或事件
3. 转折：突发状况或意外揭露
4. 博弈：各方势力的较量
5. 结果：达成某种平衡或突破
6. 余波：为后续剧情埋下伏笔

## 情绪曲线
- 开场：平静→期待
- 铺垫：自然→警觉
- 转折：震惊→紧张
- 博弈：激烈→焦虑
- 结果：释放→满足
- 余波：深思→期待
```

### 战役场景
```markdown
## 场景设置
- 地点：（平原/山地/城池/水战）
- 兵力对比：
- 天气状况：

## 战役流程
1. 铺垫：战前准备和形势分析
2. 开战：双方列阵，初次交锋
3. 僵持：战况胶着，各有胜负
4. 转折：关键决策或意外事件
5. 高潮：决定胜负的关键时刻
6. 结局：战役结果和后续影响

## 描写要点
- 宏观视角：整体战局变化
- 微观视角：士兵个体体验
- 将领视角：指挥决策过程
- 情感冲击：战争的残酷和人性

## 情绪曲线
- 铺垫：紧张→期待
- 开战：激烈→兴奋
- 僵持：焦虑→担忧
- 转折：震惊→紧张
- 高潮：震撼→激动
- 结局：释放→深思
```

