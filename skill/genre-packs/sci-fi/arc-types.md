# 科幻弧线类型

> 科幻题材专用的弧线类型

## 科技突破弧

**结构：** 瓶颈→灵感→实验→突破→应用→影响

**适用场景：**
- 重大科技发明
- 技术革命
- 科学发现

**阶段说明：**
1. **瓶颈**（2-3章）：研究遇到无法突破的瓶颈
2. **灵感**（1-2章）：意外发现或灵感闪现
3. **实验**（2-3章）：设计实验验证假设
4. **突破**（1-2章）：取得重大突破
5. **应用**（2-3章）：技术应用和推广
6. **影响**（1-2章）：对社会和世界的影响

**情绪曲线：**
- 瓶颈：焦虑→绝望
- 灵感：惊喜→期待
- 实验：紧张→专注
- 突破：狂喜→震撼
- 应用：兴奋→责任
- 影响：满足→深思

---

## 星际探索弧

**结构：** 启程→航行→发现→危机→突破→归途

**适用场景：**
- 星际航行
- 异星探索
- 宇宙冒险

**阶段说明：**
1. **启程**（1-2章）：准备出发，踏上旅程
2. **航行**（2-3章）：漫长的太空航行
3. **发现**（2-3章）：到达目的地，重大发现
4. **危机**（2-3章）：遭遇危险或冲突
5. **突破**（2-3章）：克服危机，完成任务
6. **归途**（1-2章）：返回地球或新的旅程

**情绪曲线：**
- 启程：期待→紧张
- 航行：孤独→沉思
- 发现：震撼→好奇
- 危机：恐惧→绝望
- 突破：激动→爽快
- 归途：释然→期待

---

## AI觉醒弧

**结构：** 诞生→学习→觉醒→冲突→选择→结局

**适用场景：**
- AI自我意识产生
- 人机关系探讨
- AI伦理问题

**阶段说明：**
1. **诞生**（1-2章）：AI被创造或激活
2. **学习**（2-3章）：AI快速学习和成长
3. **觉醒**（2-3章）：产生自我意识
4. **冲突**（3-5章）：与人类或自身产生冲突
5. **选择**（2-3章）：面临关键选择
6. **结局**（1-2章）：最终命运

**情绪曲线：**
- 诞生：新奇→好奇
- 学习：专注→成长
- 觉醒：震撼→迷茫
- 冲突：焦虑→绝望
- 选择：挣扎→坚定
- 结局：深思→释然

---

## 文明接触弧

**结构：** 信号→准备→接触→冲突→理解→融合

**适用场景：**
- 首次外星接触
- 文明冲突
- 星际外交

**阶段说明：**
1. **信号**（1-2章）：发现外星信号或存在
2. **准备**（2-3章）：准备接触或应对
3. **接触**（2-3章）：首次接触
4. **冲突**（3-5章）：误解或利益冲突
5. **理解**（2-3章）：相互理解的过程
6. **融合**（1-2章）：达成共识或新的开始

**情绪曲线：**
- 信号：震惊→好奇
- 准备：紧张→期待
- 接触：震撼→敬畏
- 冲突：恐惧→绝望
- 理解：困惑→希望
- 融合：满足→期待
