# 游戏弧线类型

> 游戏题材专用的弧线类型

## 升级弧

**结构：** 新手→练级→瓶颈→突破→进阶→大佬

**适用场景：**
- 等级提升
- 战力增长
- 新手到大佬的成长

**阶段说明：**
1. **新手**（2-3章）：进入游戏，熟悉系统
2. **练级**（3-5章）：快速升级，获取基础装备
3. **瓶颈**（2-3章）：遇到升级瓶颈
4. **突破**（2-3章）：找到方法突破瓶颈
5. **进阶**（2-3章）：完成进阶任务
6. **大佬**（1-2章）：成为服务器大佬

**情绪曲线：**
- 新手：好奇→兴奋
- 练级：专注→期待
- 瓶颈：焦虑→挫败
- 突破：惊喜→爽快
- 进阶：紧张→满足
- 大佬：自豪→期待

---

## 副本弧

**结构：** 发现→准备→开荒→团灭→攻略→通关

**适用场景：**
- 首次攻略副本
- 挑战高难度副本
- 团队副本

**阶段说明：**
1. **发现**（1-2章）：发现新副本
2. **准备**（2-3章）：组队和准备
3. **开荒**（2-3章）：首次尝试
4. **团灭**（1-2章）：遭遇失败
5. **攻略**（2-3章）：研究机制，调整策略
6. **通关**（1-2章）：成功通关

**情绪曲线：**
- 发现：好奇→期待
- 准备：紧张→兴奋
- 开荒：激动→紧张
- 团灭：挫败→焦虑
- 攻略：专注→希望
- 通关：爽快→满足

---

## 公会战弧

**结构：** 宣战→备战→开战→胶着→决战→胜利

**适用场景：**
- 公会战争
- 领地争夺
- 荣誉之战

**阶段说明：**
1. **宣战**（1-2章）：宣战或被宣战
2. **备战**（2-3章）：动员和准备
3. **开战**（2-3章）：正式开战
4. **胶着**（2-3章）：战况胶着
5. **决战**（2-3章）：关键战役
6. **胜利**（1-2章）：取得胜利

**情绪曲线：**
- 宣战：紧张→愤怒
- 备战：焦虑→期待
- 开战：激动→紧张
- 胶着：焦虑→担忧
- 决战：震撼→热血
- 胜利：爽快→自豪

---

## 竞技弧

**结构：** 报名→预赛→复赛→半决赛→决赛→夺冠

**适用场景：**
- 电竞比赛
- 竞技场赛季
- 巅峰对决

**阶段说明：**
1. **报名**（1章）：报名参赛
2. **预赛**（2-3章）：轻松晋级
3. **复赛**（2-3章）：遇到强敌
4. **半决赛**（2-3章）：艰难晋级
5. **决赛**（2-3章）：巅峰对决
6. **夺冠**（1章）：获得冠军

**情绪曲线：**
- 报名：期待→紧张
- 预赛：轻松→自信
- 复赛：紧张→激烈
- 半决赛：焦虑→坚持
- 决赛：震撼→激动
- 夺冠：爽快→自豪
