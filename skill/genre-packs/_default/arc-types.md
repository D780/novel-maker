# 通用弧线类型

> 适用于所有题材的基础弧线类型

## 挑战弧

**结构：** 准备→挑战→失败→成长→再战→胜利

**适用场景：**
- 升级突破
- 对抗强敌
- 完成任务

**阶段说明：**
1. **准备**（2-3章）：展示挑战的重要性，主角准备/训练
2. **挑战**（1-2章）：遭遇挑战，暴露不足
3. **失败**（1章）：失败或撤退，遭受挫折
4. **成长**（2-3章）：反思失败，寻找提升方法
5. **再战**（2-3章）：重新挑战，展示成长
6. **胜利**（1章）：最终胜利，获得奖励

**情绪曲线：**
- 准备：期待→紧张
- 挑战：紧张→激烈
- 失败：挫败→低落
- 成长：希望→坚定
- 再战：紧张→激烈
- 胜利：爽快→满足

---

## 探索弧

**结构：** 发现→探索→危险→收获→揭秘

**适用场景：**
- 秘境探险
- 副本挑战
- 寻找宝物

**阶段说明：**
1. **发现**（1-2章）：发现新地点或线索
2. **探索**（2-3章）：深入探索，收集信息
3. **危险**（1-2章）：遭遇危险或陷阱
4. **收获**（1章）：获得宝物或关键信息
5. **揭秘**（1章）：揭示更大的秘密

**情绪曲线：**
- 发现：好奇→期待
- 探索：兴奋→紧张
- 危险：恐惧→紧张
- 收获：惊喜→满足
- 揭秘：震撼→期待

---

## 冲突弧

**结构：** 摩擦→升级→对决→后果→新平衡

**适用场景：**
- 势力对抗
- 理念冲突
- 资源争夺

**阶段说明：**
1. **摩擦**（2-3章）：小规模冲突，积累矛盾
2. **升级**（2-3章）：冲突升级，双方加码
3. **对决**（2-3章）：正面交锋
4. **后果**（1-2章）：处理对决的后果
5. **新平衡**（1章）：建立新的秩序

**情绪曲线：**
- 摩擦：不满→愤怒
- 升级：紧张→激烈
- 对决：高潮→爆发
- 后果：反思→调整
- 新平衡：接受→期待

---

## 悬疑弧

**结构：** 线索→调查→误导→真相→反转

**适用场景：**
- 谜题破解
- 阴谋揭露
- 真相追寻

**阶段说明：**
1. **线索**（1-2章）：发现异常或线索
2. **调查**（2-3章）：深入调查，收集证据
3. **误导**（1-2章）：出现误导性信息
4. **真相**（1章）：揭示真相
5. **反转**（1章）：出人意料的反转

**情绪曲线：**
- 线索：好奇→困惑
- 调查：专注→紧张
- 误导：困惑→怀疑
- 真相：震惊→恍然
- 反转：震撼→深思

---

## 成长弧

**结构：** 困境→学习→实践→突破→蜕变

**适用场景：**
- 角色成长
- 能力提升
- 心态转变

**阶段说明：**
1. **困境**（2-3章）：遭遇困境，暴露不足
2. **学习**（2-3章）：寻找方法，开始学习
3. **实践**（2-3章）：实际应用，积累经验
4. **突破**（1-2章）：关键突破
5. **蜕变**（1章）：完成蜕变，获得成长

**情绪曲线：**
- 困境：挫败→低落
- 学习：希望→专注
- 实践：紧张→坚定
- 突破：激动→爽快
- 蜕变：满足→期待

---

## 关系弧

**结构：** 相遇→冲突→理解→和解→深化

**适用场景：**
- 人际关系发展
- 感情线推进
- 阵营转变

**阶段说明：**
1. **相遇**（1-2章）：初次相遇，第一印象
2. **冲突**（2-3章）：产生矛盾或误解
3. **理解**（2-3章）：逐渐理解对方
4. **和解**（1-2章）：化解矛盾
5. **深化**（1章）：关系更进一步

**情绪曲线：**
- 相遇：好奇→中立
- 冲突：对立→不满
- 理解：同情→认同
- 和解：释然→温暖
- 深化：信任→期待
