# 题材包系统

> 为不同题材提供专门的写作规则、模板和设定参考

## 使用方式

初始化小说项目时，选择题材会自动加载对应的题材包：

```
/novel-maker init 开始写一本修仙小说
```

## 可用题材包

| 题材包 | 说明 | 特定内容 |
|--------|------|----------|
| xianxia | 修仙 | 境界体系、宗门设定、秘境模板、渡劫规则 |
| urban | 都市 | 商战模板、复仇套路、崛起路径、感情线 |
| infinite-flow | 无限流 | 副本设计、强化体系、团队配置、BOSS战 |
| mystery | 悬疑 | 案件设计、推理线索、反转技巧、伏笔布局 |
| history | 历史 | 权谋模板、战役描写、改革路线、朝堂设定 |
| sci-fi | 科幻 | 科技体系、星际设定、AI描写、时间线管理 |
| game | 游戏 | 游戏系统、等级体系、装备设计、副本攻略 |
| apocalypse | 末世 | 生存体系、异能设定、势力分布、资源管理 |
| western-fantasy | 西幻 | 魔法体系、种族设定、骑士精神、王国政治 |
| wuxia | 武侠 | 武功体系、江湖规矩、侠义精神、门派纷争 |
| romance | 言情 | 感情线设计、人物关系、情感冲突、甜蜜互动 |

## 题材包内容

每个题材包包含：

- `rules.md` - 该题材的写作规则和禁忌
- `templates.md` - 该题材特有的模板（角色、场景、力量体系）
- `arc-types.md` - 该题材常用的弧线类型
- `settings.md` - 该题材的常见设定参考

## 与现有系统集成

- 初始化时选择题材，自动加载对应题材包
- 题材包中的规则补充 `genre-rules.md` 中的通用规则
- 角色创建时参考题材包中的设定模板
- 幕规划时参考题材包中的弧线类型

## 扩展题材包

如需添加新题材包，创建对应目录并包含上述四个文件即可。
