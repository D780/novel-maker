# 武侠弧线类型

> 武侠题材专用的弧线类型

## 江湖恩怨弧

**结构：** 相遇→冲突→追杀→反击→了结

**适用场景：**
- 江湖恩怨
- 复仇故事
- 恩怨情仇

**阶段说明：**
1. **相遇**（2-3章）：与仇人或恩人相遇
2. **冲突**（3-4章）：矛盾爆发，产生冲突
3. **追杀**（2-3章）：被追杀或追杀他人
4. **反击**（2-3章）：奋起反击
5. **了结**（1-2章）：恩怨了结

**情绪曲线：**
- 相遇：震惊→愤怒
- 冲突：激烈→危险
- 追杀：紧张→绝望
- 反击：愤怒→爽快
- 了结：释然→感慨

---

## 门派争霸弧

**结构：** 崛起→挑战→危机→决战→统一

**适用场景：**
- 门派争斗
- 武林争霸
- 门派统一

**阶段说明：**
1. **崛起**（3-4章）：门派逐渐崛起
2. **挑战**（3-4章）：挑战其他门派
3. **危机**（2-3章）：面临重大危机
4. **决战**（3-4章）：与最终敌人决战
5. **统一**（1-2章）：统一武林

**情绪曲线：**
- 崛起：野心→奋斗
- 挑战：自信→得意
- 危机：绝望→坚持
- 决战：激烈→爽快
- 统一：满足→感慨

---

## 武功突破弧

**结构：** 瓶颈→修炼→顿悟→突破→大成

**适用场景：**
- 武功修炼
- 境界突破
- 武学领悟

**阶段说明：**
1. **瓶颈**（2-3章）：遇到武功瓶颈
2. **修炼**（3-4章）：刻苦修炼
3. **顿悟**（1-2章）：领悟武学真谛
4. **突破**（1-2章）：突破武功境界
5. **大成**（1章）：武功大成

**情绪曲线：**
- 瓶颈：焦虑→绝望
- 修炼：刻苦→坚持
- 顿悟：惊喜→领悟
- 突破：爽快→震撼
- 大成：满足→期待

## 侠义之路弧

**结构：** 初入江湖→行侠仗义→遭遇困境→坚持信念→成为大侠

**适用场景：**
- 侠义故事
- 成长历程
- 成为大侠

**阶段说明：**
1. **初入江湖**（2-3章）：初入江湖，懵懂无知
2. **行侠仗义**（3-4章）：行侠仗义，积累名声
3. **遭遇困境**（2-3章）：遭遇重大困境
4. **坚持信念**（2-3章）：坚持侠义信念
5. **成为大侠**（1-2章）：成为真正的大侠

**情绪曲线：**
- 初入江湖：期待→紧张
- 行侠仗义：正义→满足
- 遭遇困境：绝望→低落
- 坚持信念：坚定→希望
- 成为大侠：感动→满足

---

## 武林大会弧

**结构：** 受邀→初赛→复赛→决赛→夺魁

**适用场景：**
- 武林大会
- 比武招亲
- 门派比武

**阶段说明：**
1. **受邀**（1章）：收到武林大会邀请
2. **初赛**（2-3章）：初赛阶段，展示实力
3. **复赛**（2-3章）：复赛阶段，遇到强敌
4. **决赛**（2-3章）：决赛阶段，正面对抗
5. **夺魁**（1章）：最终夺魁

**情绪曲线：**
- 受邀：期待→紧张
- 初赛：轻松→自信
- 复赛：紧张→激烈
- 决赛：高潮→爽快
- 夺魁：满足→感慨
