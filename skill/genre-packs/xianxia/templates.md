## 场景模板

### 修炼突破场景
```markdown
## 场景设置
- 地点：（修炼室/秘境/野外）
- 状态：（准备突破/正在突破/即将突破）

## 突破流程
1. 积累：灵气/资源积累到位
2. 触发：机缘或顿悟
3. 冲击：冲击境界屏障
4. 危险：走火入魔的风险
5. 突破：成功突破
6. 稳固：稳定境界

## 情绪曲线
- 积累：期待→紧张
- 触发：惊喜→专注
- 冲击：紧张→危险
- 危险：恐惧→绝望
- 突破：爽快→满足
- 稳固：平静→期待
```

### 秘境探险场景
```markdown
## 场景设置
- 秘境名称：
- 秘境等级：
- 进入条件：

## 探险流程
1. 开启：秘境开启，众人进入
2. 探索：四处探索，收集资源
3. 遭遇：遇到危险或其他修士
4. 争夺：争夺宝物或机缘
5. 收获：获得关键物品
6. 关闭：秘境即将关闭

## 情绪曲线
- 开启：期待→紧张
- 探索：好奇→兴奋
- 遭遇：紧张→危险
- 争夺：激烈→刺激
- 收获：惊喜→满足
- 关闭：紧迫→释然
```

## 力量体系模板

```markdown
## 境界划分

### 凡人境界
- 练气期：（1-9层）
- 筑基期：（初期/中期/后期/大圆满）
- 金丹期：（初期/中期/后期/大圆满）

### 仙人境界
- 元婴期：（初期/中期/后期/大圆满）
- 化神期：（初期/中期/后期/大圆满）
- 炼虚期：（初期/中期/后期/大圆满）

### 大能境界
- 合体期：（初期/中期/后期/大圆满）
- 大乘期：（初期/中期/后期/大圆满）
- 渡劫期：（初期/中期/后期/大圆满）

## 境界能力

### 练气期
- 能力：基础法术，轻身术
- 寿命：150年
- 标志：感应灵气

### 筑基期
- 能力：飞行，法术增强
- 寿命：300年
- 标志：建立丹田

### 金丹期
- 能力：金丹神通，领域雏形
- 寿命：500年
- 标志：凝聚金丹

## 升级条件

### 资源需求
- 灵石：数量随境界递增
- 丹药：辅助突破
- 功法：更高阶的修炼功法

### 时间需求
- 每个小境界：数月到数年
- 每个大境界：数年到数十年
- 越高境界，时间越长

### 突破条件
- 领悟：对天道的领悟
- 机缘：特殊机缘或宝物
- 心境：心境达到要求
```
