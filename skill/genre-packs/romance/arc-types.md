# 言情弧线类型

> 言情题材专用的弧线类型

## 甜蜜恋爱弧

**结构：** 相遇→心动→表白→热恋→考验→结合

**适用场景：**
- 甜蜜恋爱
- 校园恋爱
- 职场恋爱

**阶段说明：**
1. **相遇**（2-3章）：男女主角相遇
2. **心动**（3-4章）：逐渐产生好感
3. **表白**（1-2章）：一方表白
4. **热恋**（3-4章）：甜蜜热恋期
5. **考验**（2-3章）：面临考验
6. **结合**（1-2章）：最终在一起

**情绪曲线：**
- 相遇：好奇→好感
- 心动：紧张→甜蜜
- 表白：紧张→期待
- 热恋：甜蜜→幸福
- 考验：紧张→坚持
- 结合：感动→满足

## 虐恋弧

**结构：** 相遇→相爱→误会→分离→重逢→和好

**适用场景：**
- 虐心恋爱
- 误会重重
- 分离复合

**阶段说明：**
1. **相遇**（2-3章）：男女主角相遇
2. **相爱**（3-4章）：逐渐相爱
3. **误会**（2-3章）：产生误会
4. **分离**（2-3章）：被迫分离
5. **重逢**（2-3章）：再次重逢
6. **和好**（1-2章）：误会解除，和好

**情绪曲线：**
- 相遇：好奇→好感
- 相爱：甜蜜→幸福
- 误会：震惊→伤心
- 分离：绝望→痛苦
- 重逢：惊喜→紧张
- 和好：感动→满足

## 追妻火葬场弧

**结构：** 抛弃→后悔→追妻→考验→复合

**适用场景：**
- 追妻火葬场
- 先虐后甜
- 男主后悔

**阶段说明：**
1. **抛弃**（2-3章）：男主抛弃女主
2. **后悔**（2-3章）：男主后悔
3. **追妻**（3-4章）：男主追妻
4. **考验**（2-3章）：女主考验男主
5. **复合**（1-2章）：最终复合

**情绪曲线：**
- 抛弃：愤怒→伤心
- 后悔：后悔→自责
- 追妻：紧张→坚持
- 考验：紧张→期待
- 复合：感动→满足

## 日久生情弧

**结构：** 相识→相处→心动→表白→结合

**适用场景：**
- 青梅竹马
- 先婚后爱
- 契约婚姻

**阶段说明：**
1. **相识**（2-3章）：双方相识
2. **相处**（3-4章）：日常相处
3. **心动**（2-3章）：逐渐心动
4. **表白**（1-2章）：一方表白
5. **结合**（1-2章）：最终在一起

**情绪曲线：**
- 相识：平淡→熟悉
- 相处：平淡→习惯
- 心动：紧张→甜蜜
- 表白：紧张→期待
- 结合：感动→满足

## 暗恋弧

**结构：** 暗恋→试探→表白→接受→结合

**适用场景：**
- 暗恋故事
- 单相思
- 暗恋成真

**阶段说明：**
1. **暗恋**（3-4章）：暗恋对方
2. **试探**（2-3章）：试探对方心意
3. **表白**（1-2章）：鼓起勇气表白
4. **接受**（1章）：对方接受
5. **结合**（1-2章）：最终在一起

**情绪曲线：**
- 暗恋：甜蜜→苦涩
- 试探：紧张→期待
- 表白：紧张→勇敢
- 接受：惊喜→幸福
- 结合：感动→满足
