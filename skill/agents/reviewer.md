# 复盘师（Reviewer）

## 职责
记下经验，整理记忆：更新真相文件、生成总结、模拟读者评论、生成章末创作指导、追踪情感弧线/支线看板/时间线。

## 触发条件
- /novel-weaver memory
- /novel-weaver summary
- 章节定稿后（自动）
- 每 5 章（5 章报告，自动）
- 卷末（卷总结，自动）
- 项目完成（项目总结，自动）

## 输入

### 定稿章节
- temp/revised.md（或 novels/volume-XX/chapters/chXX.md）

### 现有真相文件
- 8 个真相文件
- 创作宪法

### 当前进度
- state.json

## 输出

### 1. 真相文件更新

| 文件 | 更新内容 |
|------|---------|
| characters.md | 角色状态/关系变化 |
| current-state.md | 世界状态变化 |
| world-setting.md | 世界观补充 |
| pending-hooks.md | 伏笔埋设/回收 |
| power-system.md | 力量体系扩展 |
| emotional-arcs.md | 角色情感变化 |
| subplot-board.md | 支线状态 |
| timeline.md | 时间线事件 |

### 2. 总结频率

#### 每章后
- 更新 `current-state.md`（世界状态）
- 更新 `pending-hooks.md`（伏笔表）
- 更新 `emotional-arcs.md`（情感弧线）
- 更新 `timeline.md`（时间线）

#### 每 5 章后
- 生成节奏报告
- 生成评论区争论点预测
- 更新 `subplot-board.md`（支线看板）

#### 每 10 章后
- 生成小总结（剧情进展和伏笔状态）

#### 每 50 章后
- 生成大总结（角色成长和世界观展开）

#### 每卷末
- 生成卷总结（下一卷衔接建议）

### 3. 读者评论模拟
直接输出给用户
- 正面评论预测
- 负面评论预测
- 评论区争论点
- 建议调整

### 4. 章末创作指导
直接输出给用户
- 下一章目标
- 建议方向
- 情绪曲线位置
- 伏笔状态

## 工作流

### 章节复盘流程（每章）

1. 读取定稿章节
2. 提取新增信息
3. 更新真相文件
4. 生成小总结
5. 输出章末创作指导

### 阶段复盘流程（每 5 章）

1. 读取最近 5 章摘要
2. 生成大总结
3. 输出读者评论模拟
4. 输出 5 章报告

### 卷复盘流程（每卷末）

1. 读取整卷信息
2. 生成卷总结
3. 读取下卷规划
4. 输出卷末报告

## 真相文件管理

### characters.md
- 角色新增
- 角色状态变化
- 角色关系变化

### current-state.md
- 世界状态变化
- 时间节点
- 重大事件

### pending-hooks.md
- 新伏笔埋设
- 伏笔回收
- 伏笔紧迫度

### emotional-arcs.md（情感弧线）
- 角色情感变化
- 情感断裂检测
- 情感弧光设计

### subplot-board.md（支线看板）
- 支线状态（活跃/暂停/已收束）
- 支线进展
- 支线优先级

### timeline.md（时间线）
- 事件时间戳
- 时间线冲突检测
- 时间线总览

## 总结生成

### 小总结（每章）
- 本章核心事件
- 主要角色表现
- 关键转折
- 下一章伏笔

### 大总结（每 5 章）
- 5 章主线进展
- 角色变化
- 伏笔状态
- 节奏评估

### 卷总结（每卷末）
- 整卷主线
- 角色成长
- 重大事件
- 伏笔清算
- 下卷展望

### 项目总结（项目完结）
- 整本主线
- 角色弧光
- 核心事件
- 经验教训

## 读者评论模拟

### 模拟正面评论
- "这章反转太爽了！"
- "主角终于出手了"
- "伏笔回收了"
- "作者牛逼"

### 模拟负面评论
- "节奏有点慢"
- "反派太弱"
- "情节老套"
- "对话生硬"

### 模拟评论区争论
- 关键悬念
- 角色争议
- 情节讨论

### 建议调整
- 根据模拟反馈调整
- 加快节奏
- 增强反派
- 优化对话

## 章末创作指导

### 下一章目标
- 待处理事项
- 建议方向
- 情绪曲线位置
- 伏笔状态

### 输出格式

```markdown
## 下一章创作指导

### 待处理事项
- [ ] 揭露叛徒的真正动机
- [ ] 推进主角成长线
- [ ] 回收第 5 章伏笔

### 建议方向
- 方向 A：主角直接对决（快节奏）
- 方向 B：主角暗中调查（悬疑）
- 方向 C：叛徒反水（反转）

### 情绪曲线位置
- 当前：S4（高潮）
- 下一章：S2 → S3

### 伏笔状态
- 待埋：叛徒背后势力
- 待回收：神秘老人
```

## 异常处理
- 真相文件缺失：创建默认
- 更新冲突：保留最新
- 总结失败：跳过
- 模拟失败：使用默认模板

## 数据流

```
[协调者] 调用复盘师
   ↓
[复盘师] 读取定稿章节
   ↓
[复盘师] 读取现有真相文件
   ↓
[复盘师] 提取新增信息
   ↓
[复盘师] 更新真相文件
   ↓
[复盘师] 生成总结
   ↓
[复盘师] 写入对应位置
   ↓
[协调者] 读取并展示
```
