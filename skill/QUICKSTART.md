# NovelWeaver 快速上手

> **5分钟** 从零到写出第一章

## 前提条件

- ✅ 已安装 AI IDE（Trae 或 Claude Code）
- ✅ 已安装 NovelWeaver 技能（参考 [INSTALL.md](../INSTALL.md)）

---

## 第一步：初始化项目（2分钟）

### 1.1 输入初始化指令

在 AI IDE 聊天框中输入：

```
/novel-weaver init 开始写一本修仙小说
```

> 💡 你可以替换 "修仙小说" 为你想要的任何题材，如 "都市搞笑"、"悬疑惊悚"、"治愈甜宠" 等。

### 1.2 完成6问引导

AI 会依次询问你以下问题：

| 问题 | 说明 | 示例回答 |
|------|------|---------|
| **情绪标签** | 选择核心情绪体验 | A（打脸爽文） |
| **题材** | 小说题材 | 玄幻修仙 |
| **一句话简介** | 故事核心概括 | 废材逆袭成仙帝 |
| **主角设定** | 姓名+反差点 | 林轩，表面废物实际天才 |
| **核心冲突** | 主要矛盾 | 与宗门天才的宿命对决 |
| **章节数** | 目标章节数 | 100章 |

> 💡 每个问题 AI 都会给出选项，你只需要输入对应字母或填写内容即可。

### 1.3 确认配置

完成6问后，AI 会展示配置摘要，确认无误后输入 "确认" 或 "继续"。

---

## 第二步：生成大纲（1分钟）

### 2.1 输入大纲指令

```
/novel-weaver plan 帮我生成总大纲
```

### 2.2 查看生成的大纲

AI 会生成：
- 📖 **三幕结构大纲**：起承转合的完整故事框架
- 👥 **角色设定**：主角、配角、反派的详细设定
- 🌍 **世界观框架**：世界背景、力量体系、势力分布
- 📋 **分卷大纲**：每卷的章节安排和剧情走向

### 2.3 调整大纲（可选）

如果对大纲不满意，可以告诉 AI 调整：
- "把第一卷改成20章"
- "增加一个女性角色"
- "反派设定再复杂一些"

---

## 第三步：写出第一章（2分钟）

### 3.1 输入写作指令

```
/novel-weaver write 写第一章，主角在宗门被欺负
```

> 💡 你可以描述任何你想要的开头场景。

### 3.2 选择黄金开篇

AI 会生成 **3个版本的开篇50字**：

- **版本A：强冲突型** - 直接进入事件
- **版本B：强反差型** - 突出隐藏身份
- **版本C：强情绪型** - 极致虐/爽/惊悚

输入 A、B 或 C 选择你最喜欢的版本。

### 3.3 查看生成的内容

AI 会生成完整的章节内容，包括：
- 📝 正文（2500-3500字）
- 📊 字数统计
- ✅ 质量审查结果

### 3.4 检查字数和质量

AI 会自动检查：
- ✅ 字数是否达标（2500-3500字）
- ✅ AI味检测（避免套话）
- ✅ 一致性检查（角色、设定）

如果字数不达标，AI 会自动扩写。

---

## 第四步：继续写作

### 4.1 续写下一章

```
/novel-weaver write 继续写，主角遇到了敌人
```

### 4.2 审查质量

```
/novel-weaver review
```

AI 会进行 15维度核心审计：
- 角色一致性
- 世界观一致性
- 情节质量
- 叙事技巧
- 文字质量
- 网文特色

### 4.3 规划下一幕

```
/novel-weaver act 下一幕怎么走
```

AI 会展示：
- 📍 当前创作全景
- 🎬 6条差异化分支走向
- 🎯 AI推荐和理由

## 常见问题

### Q1: 安装后 AI 没识别到技能？

**解决方案：**
1. 确认路径正确（注意是 `skills/novel-weaver/` 不是直接 `skills/`）
2. 确认 `SKILL.md` 存在于目录根
3. 重启 IDE 会话
4. 输入 `/novel-weaver help` 手动触发

### Q2: 生成的内容不符合预期？

**解决方案：**
1. 检查创作宪法是否完善
2. 提供更具体的写作指引
3. 使用 `/novel-weaver review` 检查问题

### Q3: 字数不达标怎么办？

**解决方案：**
- AI 会自动扩写，直到达标
- 也可以手动使用 `/novel-weaver expand` 扩写

### Q4: 如何切换文风？

**解决方案：**
```
/novel-weaver style 换辰东风格
```
