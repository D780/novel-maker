# 读者反馈模拟指南

> 复盘师使用 | 模拟读者反馈，优化创作策略

## 模拟维度

### 1. 评论区争论点预测

- **预测方法**：分析剧情冲突、角色魅力、情感共鸣点
- **争论类型**：
  - 角色站队：支持A还是支持B
  - 剧情猜测：下一步会怎样
  - 道德讨论：这样做对不对
  - 情感共鸣：太虐了/太甜了
- **输出格式**：
  - 争论点：{{debate_point}}
  - 预测热度：{{heat_level}}（高/中/低）
  - 引导建议：{{guidance}}

### 2. 读者情绪预测

- **情绪类型**：
  - 爽快感：打脸/逆袭/突破
  - 紧张感：危机/悬念/倒计时
  - 感动点：牺牲/守护/重逢
  - 愤怒感：不公/背叛/欺压
  - 期待感：伏笔/铺垫/暗示
- **输出格式**：
  - 章节：第 {{chapter}} 章
  - 预测情绪：{{emotion}}
  - 强度：{{intensity}}/10
  - 触发点：{{trigger}}

### 3. 追读率影响评估

- **评估因素**：
  - 章末钩子强度
  - 爽点密度
  - 情感投入度
  - 悬念设置
- **风险等级**：
  - 低风险：追读率稳定
  - 中风险：可能流失部分读者
  - 高风险：追读率可能大幅下降
- **输出格式**：
  - 章节：第 {{chapter}} 章
  - 风险等级：{{risk_level}}
  - 原因：{{reason}}
  - 建议：{{suggestion}}

## 评论区诱导设计

### 留白节点

- **定义**：故意不说明结果，留给读者讨论
- **示例**：
  - "他到底是谁？"
  - "她会怎么选择？"
  - "这背后隐藏着什么？"

### 讨论话题

- **定义**：设计能引发讨论的情节
- **示例**：
  - 道德困境：救一人还是救众人
  - 角色选择：A还是B
  - 剧情猜测：下一步会怎样

### 情绪触发点

- **定义**：设计能引发强烈情绪的情节
- **示例**：
  - 愤怒点：不公待遇
  - 感动点：牺牲守护
  - 爽快点：逆袭打脸

## 使用示例

### 章节完成后

1. 预测本章读者情绪
2. 识别本章争论点
3. 评估追读率影响
4. 设计下章诱导点

### 幕规划时

1. 分析当前读者画像
2. 预测剧情走向的读者反应
3. 设计留白节点和讨论话题
4. 评估整体追读率趋势
