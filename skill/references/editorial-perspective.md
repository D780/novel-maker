# 编辑视角审查

> 从网文编辑的专业视角审查作品，关注商业性和可读性

## 审查定位

编辑视角审查是审计师的扩展能力，不同于基础维度审计（关注一致性），编辑视角关注**商业价值**和**读者体验**。

## 审查维度

### 1. 开篇吸引力

**检查要点：**
- 前3章是否有足够的吸引力？
- 第1章是否建立了核心冲突？
- 金手指是否在前3章展示？
- 是否有"弃读风险点"？

**评估标准：**
- S级：第1章即有强钩子，3章内建立核心设定
- A级：前5章吸引力足够，设定展开自然
- B级：前10章有吸引力，但节奏偏慢
- C级：前10章平淡，有弃读风险

**改进建议模板：**
```
问题：前3章缺少核心冲突展示
建议：在第1章增加[具体事件]，展示主角的[核心特质]
参考：[同类作品]的开篇处理方式
```

### 2. 爽点密度与分布

**检查要点：**
- 爽点间隔是否合理？
- 爽点类型是否多样？
- 爽点强度是否递进？
- 是否有"爽点荒漠"（连续5章无爽点）？

**爽点类型清单：**
| 类型 | 说明 | 适用场景 |
|------|------|---------|
| 打脸爽 | 嘲讽→反杀 | 人际冲突 |
| 突破爽 | 升级→碾压 | 修炼/成长 |
| 逆袭爽 | 绝境→翻盘 | 危机场景 |
| 装逼爽 | 淡然→震惊 | 实力展示 |
| 复仇爽 | 血债→血偿 | 恩怨清算 |
| 智谋爽 | 布局→收网 | 权谋/推理 |
| 情感爽 | 误会→和解 | 感情线 |

**密度标准：**
- 番茄小说：每3章至少1个爽点
- 起点中文网：每5章至少1个爽点
- 晋江文学城：每5章至少1个情感爽点

### 3. 节奏曲线

**检查要点：**
- 是否有连续3章以上S4/S5（疲劳）？
- 是否有连续3章以上S1/S2（弃读）？
- 每5-8章是否有S4或S5高潮？
- 情绪单元"压-小扬-压-爆"是否执行？

**节奏评估：**
```
理想节奏：
S1→S2→S3→S4→S3→S2→S5→S1→...
（压→小扬→压→爆→缓冲→铺垫→高潮→重置）

问题节奏：
S4→S4→S4→S4（持续高潮，读者疲劳）
S1→S1→S1→S1（持续平淡，读者弃读）
```

### 4. 角色商业价值

**检查要点：**
- 主角是否有足够的"代入感"？
- 主角是否有"反差点"（表面XX实际XX）？
- 配角是否有"记忆点"？
- 反派是否有"存在感"？

**角色魅力评估：**
| 维度 | S级 | A级 | B级 | C级 |
|------|-----|-----|-----|-----|
| 代入感 | 读者即主角 | 高度共情 | 基本认同 | 无感 |
| 反差点 | 多维反差 | 核心反差 | 单一反差 | 无反差 |
| 记忆点 | 过目不忘 | 印象深刻 | 有印象 | 脸谱化 |

### 5. 钩子质量

**检查要点：**
- 每章末尾是否有钩子？
- 钩子类型是否多样？
- 钩子是否有"必须看下一章"的冲动？
- 是否有"假钩子"（看似悬念实则无关）？

**钩子强度评估：**
- S级：不看下一章睡不着
- A级：好奇后续发展
- B级：有期待但不急切
- C级：无钩子或假钩子

### 6. 追读力评估

**检查要点：**
- 读者是否有"追读"的动力？
- 是否有"未解决的悬念"？
- 是否有"期待的角色互动"？
- 是否有"未回收的伏笔"？

**追读力公式：**
```
追读力 = 悬念强度 × 角色魅力 × 爽点期待
```

### 7. 商业化潜力

**检查要点：**
- 题材是否符合平台需求？
- 是否有"改编潜力"（动漫/影视/游戏）？
- 是否有"话题性"（能引发讨论）？
- 是否有"付费点"（读者愿意付费的内容）？
