# 去AI味技巧指南

> 让AI生成的内容更像人写的，避免常见的AI写作痕迹

## 表层去AI味

### 1. 避免AI高频词汇

**禁用词汇清单：**
- "不禁"、"竟然"、"一抹"、"一丝"、"仿佛"
- "眼眸"、"嘴角"、"眉头"（过度使用）
- "淡淡"、"缓缓"、"轻轻"（连续使用）
- "仿佛"、"似乎"、"好像"（模糊表达）
- "竟然"、"居然"、"没想到"（惊讶套路）

**替换建议：**
- "不禁笑了" → 直接写"笑了"
- "一抹微笑" → "嘴角微扬" 或 具体动作
- "仿佛看到了" → "看到" 或 "眼前出现"

### 2. 避免AI句式套路

**禁用句式：**
- "他/她不禁..."
- "一股...的感觉"
- "仿佛...一般"
- "宛如...似的"
- "那一刻，他/她..."

**替换建议：**
- 使用短句、断句
- 加入口语化表达
- 使用方言或地域特色
- 加入角色特有的语气词

### 3. 避免AI描写套路

**禁用描写：**
- "阳光洒在脸上"
- "月光如水"
- "眼眸深邃"
- "嘴角勾起一抹弧度"

**替换建议：**
- 使用具体、独特的细节
- 加入角色的主观感受
- 使用比喻但避免陈词滥调
- 加入感官细节（听觉、嗅觉、触觉）

## 深层去AI味

### 1. 角色对话区分度

**核心原则：** 遮住角色名字，能认出是谁说的

**实现方法：**
- 为每个角色设计独特的说话方式
- 使用角色语音卡（见 templates/character-voice.md）
- 加入角色特有的语气词、口头禅
- 根据角色性格调整句式长短

**示例：**
```
角色A（粗犷）："干他娘的！"
角色B（文雅）："此事...不妥。"
角色C（急躁）："快！快！快！"
```

### 2. 情感表达行为化

**核心原则：** 通过行为表达情感，而非直述

**错误示例：**
- "他很愤怒"
- "她感到悲伤"
- "他心中充满了喜悦"

**正确示例：**
- "他攥紧了拳头，指节发白"
- "她低下头，眼泪无声地滑落"
- "他跳了起来，一把抱住对方"

### 3. 感官细节丰富化

**核心原则：** 不止视觉，加入其他感官

**五感清单：**
- **视觉**：颜色、形状、光影
- **听觉**：声音、音调、节奏
- **嗅觉**：气味、香味、臭味
- **味觉**：甜、苦、酸、辣
- **触觉**：温度、质感、压力

**示例：**
```
❌ "他走进了一个房间"
✅ "他推开门，一股霉味扑面而来，脚下是吱呀作响的木地板"
```

### 4. 句式长短变化

**核心原则：** 避免单调的句式长度

**节奏控制：**
- 紧张场景：短句、断句
- 舒缓场景：长句、复合句
- 对话场景：根据角色性格调整

**示例：**
```
紧张场景：
"跑！快跑！"
"来不及了。"
"完了。"

舒缓场景：
"夕阳的余晖洒在湖面上，波光粼粼，远处的山峦在暮色中若隐若现。"
```

### 5. 金句与名场面

**核心原则：** 每章至少一个"金句"或"名场面"

**金句特征：**
- 简洁有力
- 富有哲理
- 易于传播
- 符合角色性格

**名场面特征：**
- 情感高潮
- 视觉冲击
- 戏剧性强
- 读者印象深刻

**示例：**
```
金句："这个世界，弱肉强食。"
名场面：主角在绝境中反杀，说出"你以为，这就结束了？"
```

## 自检清单

每章完成后，检查以下项目：

### 词汇检查
- [ ] 是否使用了"不禁"、"竟然"、"一抹"等AI高频词？
- [ ] 是否连续使用了"淡淡"、"缓缓"、"轻轻"？
- [ ] 是否过度使用了"眼眸"、"嘴角"、"眉头"？

### 句式检查
- [ ] 是否有"他/她不禁..."的套路句式？
- [ ] 是否有"一股...的感觉"的模糊表达？
- [ ] 是否有"仿佛...一般"的陈词滥调？

### 描写检查
- [ ] 是否有"阳光洒在脸上"的俗套描写？
- [ ] 是否有"月光如水"的陈词滥调？
- [ ] 是否有"眼眸深邃"的刻板描写？

### 对话检查
- [ ] 遮住名字，能认出每个角色吗？
- [ ] 每个角色的说话方式有区分度吗？
- [ ] 对话符合角色性格吗？

### 情感检查
- [ ] 情感表达是通过行为而非直述吗？
- [ ] 是否有"他很愤怒"这样的直述？
- [ ] 情感变化有过渡吗？

### 感官检查
- [ ] 是否只有视觉描写？
- [ ] 是否加入了听觉、嗅觉、触觉？
- [ ] 感官细节是否具体？

### 节奏检查
- [ ] 句式长短是否有变化？
- [ ] 紧张场景是否使用短句？
- [ ] 舒缓场景是否使用长句？

### 亮点检查
- [ ] 本章是否有金句？
- [ ] 本章是否有名场面？
- [ ] 是否有让读者印象深刻的瞬间？

## 平台特殊要求

### 番茄小说
- 每章必须有"钩子"
- 爽点密度高（每3章至少1个）
- 节奏快，不拖沓
- 避免大段心理描写

### 起点中文网
- 世界观展开要合理
- 升级节奏要稳
- 伏笔要回收
- 避免过于直白的爽文套路

### 晋江文学城
- 情感线要细腻
- 角色魅力要突出
- 主题表达要深刻
- 避免过于商业化

## 文风库应用

使用文风库（styles/authors/）时：

1. **选择主风格**：根据题材选择1-2位作者
2. **学习特征**：阅读作者的典型作品
3. **应用特征**：
   - 句式特征（长短、节奏）
   - 词汇特征（常用词、禁用词）
   - 描写特征（视觉、听觉、心理）
   - 对话特征（语气、节奏）
4. **保持一致**：同一章节保持统一文风

## 常见问题

### Q1：为什么我的内容还是像AI写的？
**A：** 检查是否使用了AI高频词、套路句式、俗套描写。使用自检清单逐项检查。

### Q2：如何让对话更有区分度？
**A：** 为每个角色创建语音卡（templates/character-voice.md），设计独特的说话方式。

### Q3：如何避免情感直述？
**A：** 使用"情感表达行为化"技巧，通过动作、表情、语言来表达情感。

### Q4：如何增加感官细节？
**A：** 使用"五感清单"，每段描写至少包含2种感官。

### Q5：如何创造金句？
**A：** 金句要简洁、有力、富有哲理。可以从角色的核心信念出发。
