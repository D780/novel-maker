# 节奏与情绪控制系统

## 双轨机制

NovelWeaver 采用 **S1-S5节奏评级** + **情绪曲线** 双轨控制,确保张弛有度。

---

## 一、节奏评级系统 (S1-S5)

### 五级节奏定义

| 等级 | 名称 | 特征 | 适用场景 | 频率建议 |
|------|------|------|---------|---------|
| S5 | 极高潮 | 生死关头、终极对决 | 卷末、大高潮 | 每卷1-2次 |
| S4 | 高潮 | 战斗、突破、打脸 | 关键冲突 | 每10章1次 |
| S3 | 上升 | 准备、计划、推进 | 高潮前铺垫 | 常用 |
| S2 | 平缓 | 日常、对话、探索 | 高潮后缓冲 | 常用 |
| S1 | 低谷 | 纯说明、设定介绍 | 开篇、过渡 | 偶尔 |

### 节奏规则

- ❌ 不能连续3章S4或S5 (读者疲劳)
- ❌ 不能连续3章S1或S2 (读者弃读)
- ✅ 每5-8章必须有S4或S5
- ✅ 每3章必须有S3或更高

### 理想节奏曲线

```
S2 → S3 → S4 → S2 → S1 → S3 → S4 → S5 → S2 → S3...
  ^       ^         ^           ^
  铺垫    上升      小高潮      大高潮
```

### 卷级节奏分布

| 阶段 | 章节范围 | 节奏走向 | 内容 |
|------|---------|---------|------|
| 起 | 1-5章 | S1→S2→S3 | 引入新事件/场景 |
| 承 | 6-15章 | S2→S3→S4 | 发展冲突 |
| 转 | 16-25章 | S3→S4→S4 | 中点转折 |
| 合 | 26-35章 | S4→S5→S2 | 高潮+余韵 |

## 二、情绪曲线系统

### 六大情绪标签

| 标签 | 核心体验 | 节奏要求 | 爽点间隔 |
|------|---------|---------|---------|
| 打脸爽文 | 扮猪吃虎、逆袭碾压 | S4-S5密集 | ≤3章 |
| 极致虐恋 | 追妻火葬场 | S2→S4波动大 | 5-8章 |
| 爆笑反套路 | 沙雕吐槽、神转折 | S3为主 | 3-5章 |
| 悬疑惊悚 | 细思极恐、烧脑 | S2→S3→S5阶梯 | 5-8章 |
| 治愈甜宠 | 日常温馨、双向奔赴 | S1-S2为主 | 3-5章 |
| 脑洞大开 | 系统流、末日囤货 | S3-S4稳定 | 5-10章 |

### 核心公式: 每3章"压-小扬-压-爆"循环

| 章节 | 情绪阶段 | 读者心理 | S级对应 |
|------|---------|---------|--------|
| 第1章 | 压 | "太憋屈了,我要看他翻身" | S2-S3 |
| 第2章 | 小扬 | "果然有反转!" | S3-S4 |
| 第3章 | 压→爆预告 | "下章必须打脸!" | S3-S5 |

> 第4-6章重复此循环,爆点规模逐轮扩大。

### 六类情绪标签节奏模板

**1. 打脸爽文**
- 第1章(S2): 主角因隐藏身份被嘲讽/欺负
- 第2章(S3): 不经意展露实力,但被误认为运气
- 第3章(S4): 遭遇更强对手,章末暗示即将当众揭穿

**2. 极致虐恋**
- 第1章(S2): 女主被男主虐心/忽视
- 第2章(S3): 女主觉醒,做出离开决定
- 第3章(S4): 男主追悔莫及,章末出现后悔画面

**3. 爆笑反套路**
- 第1章(S2): 主角陷入经典套路剧情
- 第2章(S3): 用清奇脑回路破解套路
- 第3章(S3): 系统试图纠正,章末留"这也能行?"

**4. 悬疑惊悚**
- 第1章(S2): 遭遇无法解释的诡异事件
- 第2章(S3): 找到一点线索或规律
- 第3章(S4): 发现规律是陷阱,章末留细思极恐钩子

**5. 治愈甜宠**
- 第1章(S1): 主角处于孤独/缺爱状态
- 第2章(S2): 与CP产生温暖互动
- 第3章(S3): 出现小误会,章末CP做出心动举动

**6. 脑洞大开**
- 第1章(S2): 获得金手指但面临生存危机
- 第2章(S3): 利用金手指解决小危机
- 第3章(S4): 发现金手指有代价,章末出现更大机遇

### 单章内部情绪拐点 (2200-2800字内至少3次变化)

| 位置 | 字数 | 功能 |
|------|------|------|
| 开头 | 50-250字 | 延续或急转,快速建立本章情绪基调 |
| 中间 | 1500-1900字 | 情绪逐步升级,至少一次小起伏 |
| 结尾 | 450-650字 | 情绪到达本章峰值后急转,留下钩子 |

### 情绪拐点句式库

| 情绪转换 | 示例 |
|---------|------|
| 憋屈→暗爽 | "他低着头,嘴角却微微上扬——他们还不知道,自己惹了谁。" |
| 震惊→愤怒 | "直到看到那张照片,她才发现,这三年的眼泪都流给了骗子。" |
| 绝望→希望 | "门锁转动的声音响起时,她握紧了水果刀。但进来的,是那张熟悉的脸。" |
| 得意→恐惧 | "他以为自己赢了,直到转身看见那个人的眼睛。" |
| 平静→震惊 | "我像往常一样打开门,然后看见了这辈子最不想看见的人。" |

## 三、Hook点与爽点管理

### Hook点密度

| 章节位置 | Hook数量 | 类型 |
|---------|---------|------|
| 开篇章 | 2-3 | 引入悬念 |
| 发展章 | 1-2 | 推进悬念 |
| 高潮章 | 1-2 | 结果悬念 |
| 过渡章 | 1-2 | 新事件引子 |

**Hook冷却规则**:
- 同类Hook不能连续使用
- 每5章必须有Hook兑现
- 未兑现的Hook不能超过10章

### 爽点密度参考

| 题材 | 爽点间隔 | 爽点类型 |
|------|---------|---------|
| 爽文 | 每2-3章 | 打脸、逆袭、突破 |
| 悬疑 | 每5-8章 | 解谜、反转 |
| 言情 | 每3-5章 | 感情推进、误会解除 |
| 凡人流 | 每5-10章 | 突破、收获 |

### 信息密度控制

| 信息类型 | 每章数量 | 说明 |
|---------|---------|------|
| 新设定 | 0-2 | 不能过多新设定 |
| 新角色 | 0-2 | 不能同时引入太多角色 |
| 情节进展 | 1-2 | 必须有实际进展 |
| 伏笔 | 1-3 | 埋设或回收 |

## 四、评论区诱导设计

在以下节点刻意留白,引发读者互动:

1. **两难选择**: 主角在两个都有道理的选项中犹豫,章末不写决定
2. **信息差**: 让读者比主角多知道一点信息,但不全知道
3. **误解不解释**: 主角被误解时不立刻解释
4. **配角高光**: 给反派或配角一个令人同情的瞬间
5. **规则争议**: 主角做出道德灰色地带的选择,不评价对错

## 五、检查清单

### 章节级检查

- [ ] 本章是否包含至少3次情绪变化?
- [ ] 本章结尾钩子是否比上一章更强或持平?
- [ ] 本章在3章单元中是否完成了相应情绪任务?
- [ ] 是否有至少一处让读者想"划线评论"的情绪高潮句?

### 五章节奏报告模板

```
## 第X章-第Y章 节奏报告
- 情绪曲线总览: [压-小扬-压-爆]完成度
- 爆点密度: [S4+次数/5章]
- 评论区争论点预测: [最可能引发讨论的2-3处]
- 后续建议: [调整情绪标签权重/为高人气配角加戏/强化某类Hook]
```

## 六、自动优化建议

| 检测到问题 | 建议 |
|-----------|------|
| 连续3章S4+ | 下一章安排日常缓冲 |
| 连续3章S2- | 下一章安排冲突/高潮 |
| Hook超5章未回收 | 尽快兑现或说明原因 |
| 无爽点超过5章 | 安排打脸/逆袭/突破 |
| 信息密度过高 | 拆分到后续章节 |
| 情感单一 | 安排情感对比 |
